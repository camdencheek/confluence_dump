Retirement Notification

The NCI PID data portal has been retired. PID data are now available for the research community via the NDEx database, hosted by the Ideker Lab at the UC San Diego School of Medicine. To access PID files click “start now” and on the public server welcome page type nci in the search box. NDEx will return all the PID network files. PID data is also available for download in BioPax format on GitHub .  If you have any questions, please contact the NCI CBIIT Application Support team.
