The purpose of this page is to list projects that are, or potentially might, submit data to ICDCPossible submission status values:
  • Identified - Possible submission but no contact with project owners
  • Preliminary - Contact with project owners, discussions underway
  • Declined - Project decided not to submit to ICDC
  • Pre-approved - Project has submitted application to Data Governance Advisory Board
  • Approved - DGAB has approved, awaiting submission
  • Hold - no data is actively being submitted
  • Submitting - Project is actively submitting data to ICDC
  • Under Embargo - Submission done, awaiting final project approval for release
  • Released - Data are available for public access
  • Closed- No additional data will be submitted
