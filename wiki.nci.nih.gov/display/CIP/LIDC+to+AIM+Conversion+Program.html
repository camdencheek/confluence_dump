If you have not installed JDK in your machine, you can download and install the software by clicking here . To run conversion program, download the jar file from Attachment of this page first.  Then open a dos window, go to the same directory of the jar file is saved and type "java -jar aim-conversion_fat.jar LIDC <LIDC annotation file path and name> <output directory name>" For example:  java -jar aim-conversion_fat.jar LIDC "c:\aim\testData\LIDC\https-76350.xml" "C:\AIM\testData\LIDC\toAim"
