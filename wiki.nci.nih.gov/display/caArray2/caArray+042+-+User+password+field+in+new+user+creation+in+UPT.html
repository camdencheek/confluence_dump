Is "user password" a required field for new user creation in UPT?

Topic: caArray and UPT Integration Release: Up to caArray 2.x Date entered: 01/10/2010

Answer

If you are planning to login to caArray with LDAP Authentication, the "User Password" field should be left as blank. (For more details, see caArray 021 - Configuring caArray for LDAP authentication , Login to caArray with LDAP Authentication.) If you are going to login with a Username/Password, a Not Null value must be entered for the "User Password" field.
