Question: Are there secure Grid services for caArray?

Topic: caArray Installation and Upgrade Release: caArray 2.4.0 Date entered: 04/15/2011

Details about the Question

I notice that on the "caArray 2.4.0 install.properties" wiki page , there are following properties: grid.ssl.enable grid.ssl.keystore.location grid.ssl.keystore.pass grid.ssl.keystore.alias grid.ssl.fullyqualified.hostname grid.secure.enable grid.secure.port grid.secure.cert.location grid.secure.key.location Do these mean that there's secure Grid services for caArray? Related forum Posting

Answer

No. These parameters were included due to requirements from BDA templates. However, caArray does not use the parameters because it does not yet have secure grid service implementation. It is important to note that these parameters should not be removed or modified because the installer checks the existence of the parameters. It is also worth noting that removing and editing of these parameters was not tested.
