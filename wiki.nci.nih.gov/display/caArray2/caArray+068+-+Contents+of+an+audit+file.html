Question: What can we find in an audit file?

Topic: caArray Usage Release: Up to caArray 2.x Date entered: 10/18/2010

Answer

The server.log, located under $JBOSS_HOME\server\default\log, captures many user activities on the caArray instance. The table below summarizes what you can find in the audit file. What is inside the caArray Audit File?
File Type Execution In Audit File?
Experiment Create Yes
Experiment Delete N/A
Array Data File Insert Yes
Array Data File Edit No
Array Data File Delete No
Array Design File Insert No
Array Design File Edit No
Array Design File Delete No
