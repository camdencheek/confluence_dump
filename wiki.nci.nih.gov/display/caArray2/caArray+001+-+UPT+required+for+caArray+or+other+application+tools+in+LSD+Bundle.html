Question: Is UPT required for caArray or other application tools in the LSD Bundle?

Topic: caArray and UPT Installation Release: Up to caArray 2.2.0 Date entered: 12/18/2008

Answer

Yes. The User Provisioning Tool (UPT) is an LSD user management tool that provides the authorization policy to all LSD applications. To configure the application properly, UPT has to be properly installed prior to any other LSD application installation.The Life Sciences Distribution (LSD) applications use Common Security Module (CSM) APIs to enforce Authorization Policy. In their business logic applications make CSM API calls to check if users have access and permissions before providing any access to the underlying data. The Authorization policy is stored in the CSM Schema and provisioned by the CSM User Provisioning Tool (UPT). The LSD bundle is architected with the capability to share a single database and single UPT for all applications.The diagram below is a simplified illustration of two components used in the caArray application: caArray and UPT. The caArray application runs in its own container (application server). UPT runs in a separate container. Both caArray and UPT have their own CSM schema. caArray user information will be created and maintained with UPT, but stored in the caArray CSM schema.Diagram of caArray Components
