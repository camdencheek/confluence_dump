Question: What is the version compatibility between geWorkbench and caArray?

Topic: caArray and geWorkbench Integration Release: caArray 2.x Date entered: 09/30/2009

Answer

geWorkbench can remotely retrieve data from caArray via the caArray API. The upcoming caArray 2.3.0 API will establish backward compatibility in the future but does not work with geWorkbench versions earlier than the upcoming 1.8.0 release. The table below summarizes the compatibility between the different versions of caArray and geWorkbench (versions in a row are compatible with each other).
geWorkbench Version caArray Version caBio Comments
1.8 2.3 4.2 Upcoming releases for geWorkbench and caArray
1.7 2.1,2.2.1 4.2 Current and previous versions for geWorkbench and caArray
1.5.1-1.6.3 2.1,2.2.1 4.0,4.1 Past releases
1.5.0 2.0 4.0 Past releases
