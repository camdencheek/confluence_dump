Should my caArray grid service be Analytical or Data Service?

Topic: caArray Installation and Upgrade Release: Up to caArray 2.x Date entered: 01/12/2010

Details about the Question

I went to http://cagrid-portal.nci.nih.gov/ to search for my caArray grid service. I see that it appears as an Analytical Service, not as a Data Service. I am assuming caArray is a data service. Is there a setting I have to set?

Answer

Starting with caArray 2.3.0.1, the caArray grid node is automatically classified as an Analytical Service. All the grid nodes from the previous caArray versions were classified as a Data Service. The user does not need to set any parameter.
