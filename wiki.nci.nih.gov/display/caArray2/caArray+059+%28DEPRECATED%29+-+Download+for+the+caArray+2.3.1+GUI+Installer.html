Question: Where do I download the caArray 2.3.1 GUI Installer?

Topic: caArray Installation and UpgradeRelease: caArray 2.3.1Date entered: 05/14/2010

Answer

The GUI-based installer: caarray_gui_distribution_2_3_1_1.jar for caArray version 2.3.1.1 can be downloaded by following the steps below:For browser based download:
  1. Go to: ftp://caftpd.nci.nih.gov/pub/
  2. If there is any login required, use username = anonymous
  3. Select folder: caARRAY
  4. Right click on the file caarray_gui_distribution_2_3_1_1.jar
  5. Select "Save Link As" to save the file
The full path for the file is: caftpd.nci.nih.gov/pub/caARRAY/caarray_gui_distribution_2_3_1_1.jar For command line based ftp client download:
  1. FTP to ftp server: caftpd.nci.nih.gov with the command: FTP caftpd.nci.nih.gov
  2. If there is any login required, use username = anonymous
  3. open folder: pub with command: cd pub
  4. open folder: caARRAY with command: cd caARRAY
  5. List the content of the caARRAY directory: ls -alt
  6. Download the GUI installer by using command:
get the file *caarray_gui_distribution_2_3_1_1.jar*
