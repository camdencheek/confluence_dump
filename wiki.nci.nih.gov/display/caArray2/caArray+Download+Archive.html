Primary Distribution

caArray 2 provides a simplified approach to installation on Linux and Windows platforms. Much of the installation process has been automated such that, if you accept the default values, you can install caArray by typing a single command. With the exception of the database, all supporting tools are automatically installed and configured as part of the installation. The installation provides additional diagnostics to better ensure the efficacy of the installation. The installation package contains the binaries and scripts necessary to install the application, API, and grid service.For caArray users who already installed caArray 2.3.0.1 – 2.3.1.1, an upgrade script is provided. This script will upgrade the earlier 2.3.x caArray instance to 2.4.1 while retaining the data and the local configuration setting.For caArray users who already installed caArray 2.0.0 – 2.2.1, use the caArray 2.3.0.1 upgrade script first, then upgrade caArray 2.3.0.1 to 2.4.1.A GUI version as well as a command-line version is provided for the install/upgrade.
  • caArray Install command-line Script (caarray_install_2_4_1_0.zip)
  • caArray Upgrade command-line Script (caarray_upgrade_2_4_1_0.zip)
  • caArray GUI Installer and Upgrader (caarray_gui_distribution_2_4_1_0.jar)
