Question: Are there any guidelines for applying OS patching and Java updates on servers running caArray?

Topic: caArray Installation and Upgrade Release: Up to caArray 2.x Date entered: 05/20/2010

Details about the Question

We have caArray 2.1.1 running on a Windows server. Are there any guidelines for applying OS patching and Java updates on servers running caArray?

Answer

Every released caArray application has been tested on a supported tech stack. There is a possibility that caArray might work on a platform besides the supported one, but the caArray team cannot support the updates to the OS or Java outside of what has been tested. The supported tech stack for the latest version of caArray as of this writing, v2.4.0.1, can be found in the latest caArray Installation Guide . ***Tested Environment for caArray 2.4.0.1*** The caArray 2.4.0.1 installation has been tested on
  • Linux Red Hat Enterprise,
  • Linux AS 4 64/32-bit (for AMD chipset)
  • Windows XP/2003 environments.
  • Java 2 Platform Standard Edition 5.0 Update 10 (J2SE 5.0)
  • MySQL v5.0.45
  • Apache Ant v1.7.0 (for installs and upgrades)
While the installation may work in other Linux and Windows environments, it has only been tested in the environments listed above.
