caArray File Loading Process

Topic: caArray Usage Release: caArray 2.0 and above Date entered: 04/20/2009

Description of the caArray File Loading Process

There are many types of files that can be uploaded into caArray. We have developed a series of knowledge base entries to explain the different aspects of the file uploading process in caArray. The topics in this series cover:
  • caArray 014 - Which files can be uploaded into caArray? This entry summarizes which files can be uploaded and how to upload them.
  • caArray 016 - Which file can be parsed into caArray? What is the benefit of file parsing? This entry explains what file parsing means in caArray and which types of the files will be parsed.
  • caArray 017 - What is the meaning of the caArray Status of Importing - Imported versus Imported Not Parsed? This entry explains why different files are in different states in caArray. It also gives a detailed illustration on how files are processed by caArray and how they end in different states.
  • caArray 018 - What causes the error "Failed Validation"? Before a file is parsed into caArray, it has to pass caArray's validation process. This entry explains what the file validation process is looking for.
