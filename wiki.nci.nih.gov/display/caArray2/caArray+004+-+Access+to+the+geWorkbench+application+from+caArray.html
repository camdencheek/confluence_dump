Question: Can I access the geWorkbench application from caArray?

Topic: caArray and geWorkbench Integration Release: Up to caArray 2.2.0 Date entered: 01/06/2009

Answer

No. You can not access the geWorkbench application from the caArray web interface. However, you can access caArray data from the geWorkbench application. For detailed information, please view the geWorkbench tutorial on remote data access .
