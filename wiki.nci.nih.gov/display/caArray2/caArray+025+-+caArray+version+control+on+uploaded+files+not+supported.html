Question: Does caArray have version control on uploaded files?

Topic: caArray Usage Release: caArray 2.x and above Date entered: 06/25/2009

Answer

No. Version control on Uploaded Files is not supported in caArray. caArray does not allow users to upload a file with the same name that already exists in the experiment. To re-load the same file, the user need to either delete the file in caArray or load the file under a different name.
