After you log in (as described in Logging In via the GUI ), the Dashboard appears. On this page, the My Notifications section lists the notifications to which you have subscribed. For details on subscribing, refer to Subscribing to Notifications . The following table describes the actions you can take in the list of notifications.
To do this ... Do this ...
Sort the list Click any column header. To reverse the sort, click it again. To clear the sort, click it a third time.
Filter the list Click a text box in a column header. Type a value. The system displays only the results that contain the specified value.
View details about an item Click the notification ID. The system displays the details page for that notification.
