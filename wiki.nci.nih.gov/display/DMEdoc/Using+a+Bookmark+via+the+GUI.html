If your user account has the Read, Write, or Own permission level on a collection or data file of interest in DME, and if you have previously created a bookmark to access that collection or data file (as described in Creating a Bookmark via the GUI ), you can use that bookmark. To use a bookmark:
  1. Browse for content as described in Browsing for Data via the GUI . The Browse page appears.
  2. Click the bookmarks icon ( ), visible on the left side of the page. The Bookmarks dialog box appears.
  3. Click the bookmark of interest. The bookmarked item appears. If it is a collection, it appears in the Browse page. If it is a data file, it appears in the Data File page.
From the Browse page, you can perform various actions. For instructions, refer to the following pages:
  • Viewing Metadata via the GUI
  • Creating an Empty Collection via the GUI
  • Registering from Your Local System via the GUI
  • Registering Data in Bulk via the GUI
  • Updating Metadata via the GUI
  • Deleting a Bookmark via the GUI
