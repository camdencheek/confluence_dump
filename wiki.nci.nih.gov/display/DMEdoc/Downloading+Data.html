If your user account has the Read, Write, or Own permission level on a collection of data files or a single data file in DME, you can download that data using the web application (GUI) or Command Line Utilities (CLU).
  • If you want fine-grained control of the activity, or if you want to integrate it into your scientific workflow, consider using CLU.
  • Otherwise, consider using the GUI.
Before downloading data, you can view the file size of a collection in the following ways:
  • Perform a detailed search for collections, as described in Searching for Data via the GUI (or run a saved detailed search from the Dashboard). As you select collections in the search results page, the total size appears above the table.
  • Generate a collection path report, as described in Viewing a Base Path Report .
  • Browse for the data, as described in Browsing for Data via the GUI . The top of the detail panel displays the size of the selected collection.
