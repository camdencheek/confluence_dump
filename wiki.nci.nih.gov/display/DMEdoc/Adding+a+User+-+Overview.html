If your user account has the group administrator or system administrator role, you can set up a new user, which involves the following activities:
  1. Register an NIH user account in the system, which allows that user to use DME.
  2. Give that user access to data.
  3. Create a bookmark for that user to access the data.
In some cases, you can choose whether to perform each activity using the web application's graphical user interface (GUI) or Command Line Utilities (CLU).
  • If you want fine-grained control of the activity, or if you want to integrate it into your scientific workflow, consider using CLU.
  • Otherwise, consider using the GUI.
The following table compares the functionality available in each interface:
Activity GUI CLU
Add the user. Adding a User via the GUI Registering a User via the CLU or Adding a Bookmark and Permissions via the CLU
Give that user access to data. Sharing Data via Permissions Adding a Bookmark and Permissions via the CLU
Create a bookmark for that user. (In the GUI, you can create a bookmark only for yourself.) Adding a Bookmark and Permissions via the CLU
