If your user account has the group administrator or system administrator role, you can view a Base Path report.
  1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. Click the Reports tab. The Reports page appears.
  3. From the Report Type list, select Base Path Report.
  4. In the Base Path field, type a value. The system displays only the results that contain the specified value. Select a path.
  5. (Optional) Adjust the date range in the From Date and To Date fields.
  6. (Optional) Specify whether to Include Archive Summary in the report.
  7. Click Generate. The report appears.
The following table describes the data available in the report:
Data Description
Total Registered Users The total number of users assigned to the selected base path as the default base path. For example, /FNL_SF_Archive as a base path might have 455 total registered users.
Total Archived Data Size (Bytes) The total size of the archived data in the specified location:
  • The first number is in bytes.
  • The second number (in parentheses) is in SI units.
Archive Summary A summary of the data. Each line provides the storage provider (vault), name of the storage bucket, and the size of the data within it.
Largest Archived Data File Size (Bytes) The size of the largest archived data file in the specified location:
  • The first number is in bytes.
  • The second number (in parentheses) is in SI units.
Average Archived File Size (Bytes) The average size of an archived data file in the specified location:
  • The first number is in bytes.
  • The second number (in parentheses) is in SI units.
Total Number of Data Objects Archived The total number of archived data files in the specified location.
Total Number of Collections The total number of collections in the specified location, subdivided by depth in the hierarchy. For example, the collection path /FNL_SF_Archive/PI_Lab_JaneDoe might have one collection at the PI_Lab level, six collections at the Project level, and two collections at the Flowcell level.
Files <with specified size> The total number of archived data files in the specified location, subdivided by size.
