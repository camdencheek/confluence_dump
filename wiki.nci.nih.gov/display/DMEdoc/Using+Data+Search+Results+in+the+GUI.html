After you perform a search for data (as described in Searching for Data via the GUI ), the search results page appears. The following table describes the actions you can take on the search results page.
To do this ... Do this ...
Save search criteria Click Save Search Criteria. The Save Search Criteria dialog box appears. For details, refer to Saving a Search via the GUI .
Edit search criteria Click Edit Search Criteria. The Search page appears. For details, refer to Searching for Data via the GUI .
Sort the list Click any column header. To reverse the sort, click it again. To clear the sort, click it a third time.
Filter the list Click a text box in a column header. Type a value. The system displays only the results that contain the specified value.
Export collection or file search results Click the menu icon ( ) to the right of all columns. An option list appears. For further instructions, refer to Exporting Search Results via the GUI .
View details about an item Click the collection or file path. The system displays the details page for that collection or file. For details, refer to Viewing Metadata via the GUI .
Download a collection or file In the row for the collection or file of interest, click the download icon ( ). The Download page appears. For details, refer to Downloading Data via the GUI .
Download multiple files In the selection column, select each file of interest. Then click the download icon above the list. The Download page appears. For details, refer to Downloading Data via the GUI .
Download multiple collections In the selection column, select each collection of interest. Then click the download icon above the list. The Download page appears. For details, refer to Downloading Data via the GUI .
Create a link to a file In the row for the file of interest, click the link icon ( ). The Create Link page appears. For details, refer to Creating a Link via the GUI .
Create multiple file links In the selection column, select each source file and click the link icon above the list. The Create Link page appears. For details, refer to Creating a Link via the GUI .
Manage permissions for a collection or file In the row for the collection or file of interest, click the permission icon ( ). The permissions page for that collection or file appears. For details, refer to Managing Permissions Individually via the GUI .
Manage permissions for multiple collections or files In the selection column, select the each collection or file of interest. Then click the permission icon above the list. The bulk permissions page appears. For details, refer to Managing Permissions in Bulk via the GUI .
Move to the first page of results Click the Page to first icon ( ).
Move to the previous page of results Click the Page back icon ( ).
Move to the next page of results Click the Page forward icon ( ).
Move to the last page of results Click the Page to last icon ( ).
Select the number of items per page In the items per page box, select a number. The minimum number of items per page is 10 and the maximum is 5,000.
