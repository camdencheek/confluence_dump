This user guide describes how to perform a variety of tasks in DME. In some cases, you can choose whether to perform each activity using the web application's graphical user interface (GUI) or Command Line Utilities (CLU).
  • If you want fine-grained control of the activity, or if you want to integrate it into your scientific workflow, consider using CLU.
  • Otherwise, consider using the GUI.
The following table compares the functionality available in each interface:
Task GUI CLU
Getting Started Getting Started with DME GUI Getting Started with DME CLU
Finding Data Finding Data via the GUI Finding Data via the CLU
Using Bookmarks Creating a Bookmark via the GUI Using a Bookmark via the GUI Deleting a Bookmark via the GUI Adding a Bookmark via the CLU
Viewing or Retrieving Metadata Viewing Metadata via the GUI Retrieving the Metadata of a Collection via the CLU Retrieving the Metadata of a Data File via the CLU
Registering Data Registering Data via the GUI You can register data from the following sources:
  • Local system (one file up to 10 GB)
  • Globus
  • AWS S3
  • Google Drive
  • Google Cloud storage
Registering Data via the CLU You can register data from the following sources:
  • Local system
  • Globus (one file)
  • AWS S3 (one file)
  • Google Cloud storage (one file)
Downloading Data Downloading Data via the GUI You can download data to the following destinations:
  • Local system (one file)
  • Globus
  • AWS S3
  • Google Drive
  • Google Cloud storage
Downloading Data via the CLU You can download data to the following destinations:
  • Local system (one file)
  • Globus
  • AWS S3
Viewing Reports Viewing Reports (Report functionality is not available in the CLU.)
Subscribing to Notifications Subscribing to Notifications (Notification subscription functionality is not available in the CLU.)
Managing Permissions Sharing Data via Permissions Adding a Bookmark and Permissions via the CLU
Managing Users and Groups Managing Users and Groups via the GUI Managing Users via the CLU
Deleting Data Deleting Data via the GUI Deleting Data via the CLU
