After you click Manage tab > Registration Tasks (as described in Viewing Registration Status ), the Registration Tasks page appears, listing all ongoing and completed requests for the transfer of collections or data files to DME. Similarly, after you click Manage tab > Download Tasks (as described in Viewing Download Status ), the Download Tasks page appears, listing all ongoing and completed requests for the transfer of collections or data files from DME. The following table describes the actions you can take on the Registration Tasks page or Download Tasks page.
To do this ... Do this ...
Sort the list Click any column header. To reverse the sort, click it again. To clear the sort, click it a third time.
Filter the list Click a text box in a column header. Type a value. The system displays only the results that contain the specified value.
View details about an item Click the Task ID. The system displays the details page for that download or registration task. For details, refer to the following pages:
  • Viewing the Details of a Download Task
  • Viewing the Details of a Registration Task
Move to the first page of results Click First.
Move to the previous page of results Click Previous.
Move to the next page of results Click Next.
Move to the last page of results Click Last.
