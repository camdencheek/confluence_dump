If your user account has the Read, Write, or Own permission level on a collection, you can download that collection and all subcollections to your local system. The following table compares the functionality available with each command:
Command Quantity Destination
dm_download_collection 1 collection Local system
dm_download_collection_globus 1 or more collections Globus
dm_download_collection_s3 1 or more collections AWS S3
