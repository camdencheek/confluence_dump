If your user account has the group administrator or system administrator role, you can view a Data Owner report.
  1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. Click the Reports tab. The Reports page appears.
  3. From the Report Type list, select Data Owner Report.
  4. Click Generate. The report appears. For additional instructions, refer to Filtering, Exporting, and Navigating Reports .
The following table describes the data available in the report:
Data Description
DOC A division, office, or center (DOC).
Base Path One of the base paths for that DOC.
Collection Path One of the collection paths in that base path.
Data Owner Data owner of the collection.
Data Generator Data generator for the collection.
Collection Size (Bytes) The size of the collection, in bytes.
Collection Size The size of the collection, in SI units.
Archive Summary (Vault, Bucket, Size) To display a summary of the data for a collection, click the information icon. Each line provides the storage provider (vault), name of the storage bucket, and the size of the data within it. (An export of this report contains only the displayed values.)
