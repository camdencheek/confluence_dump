Sometimes a download fails due to network issues or other issues. Sometimes, you might decide to cancel a download in progress, as described in Viewing the Details of a Download Task . In these cases, consider the following steps:
  1. Check whether the Globus group has been given write permission to the target Globus endpoint. For instructions, refer to Preparing to Use Globus .
  2. Contact NCIDataVault@mail.nih.gov  with details about the problem.
  3. If the issue persists, view the details about a specific failed download request, as described in Viewing the Details of a Download Task .
  4. Click the Retry icon ( ), visible on the right or left side of the page. The system submits a new request, identical to the original request. A message appears with a Task ID link for the new request.
  5. Click the link to view the new download status row on the Download Tasks page.
