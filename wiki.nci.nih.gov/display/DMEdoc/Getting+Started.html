This user guide describes how to perform various activities in DME. This page family provides introductory guidance, depending on how you intend to use DME:
  • If you need to move a large volume of data (more than 10 GB), you can do so using a Globus endpoint, Google Drive, or an Amazon Web Services (AWS) S3 bucket. DME allows you to use Globus, Google Drive, or S3 for high performance transfer of large files.
  • You can choose whether to use the web application's graphical user interface (GUI) or Command Line Utilities (CLU).
    • If you want fine-grained control of the activity, or if you want to integrate it into your scientific workflow, consider using CLU.
    • Otherwise, consider using the GUI.
