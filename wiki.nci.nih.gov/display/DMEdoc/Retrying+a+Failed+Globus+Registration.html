Sometimes a registration fails due to network issues or other issues. In these cases, consider the following steps:
  1. Check whether the Globus group has been given read permission to the Globus source. For instructions, refer to Preparing to Use Globus .
  2. Check whether the file or files exist, as specified in the registration request.
  3. Contact NCIDataVault@mail.nih.gov  with details about the problem.
  4. If the issue persists, view the details about the specific failed registration request, as described in Viewing the Details of a Registration Task .
  5. Click the Retry icon ( ), visible on the right or left side of the page. The system submits a new request, identical to the original request. A message appears with a Task ID link for the new request.
  6. Click the link to view the new registration status row on the Data Registration Task Details page.
