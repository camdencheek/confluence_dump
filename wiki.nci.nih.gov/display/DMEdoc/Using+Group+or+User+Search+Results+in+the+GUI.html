After you perform a search for users or groups (as described in Finding a User via the GUI or Finding a Group via the GUI ), the search results list appears. The following table describes the actions you can take in the search results list.
To do this ... Do this ...
Sort the list Click any column header. To reverse the sort, click it again. To clear the sort, click it a third time.
Filter the list Click a text box in a column header. Type a value. The system displays only the results that contain the specified value.
Edit an item You can view details in the following lists:
  • In the user search results, click Edit. The Update User dialog box appears. For details, refer to Updating a User via the GUI .
  • In the group search results, click Edit. The Update Group page appears. For details, refer to Updating a Group via the GUI .
Move to the first page of results Click the Page to first icon ( ).
Move to the previous page of results Click the Page back icon ( ).
Move to the next page of results Click the Page forward icon ( ).
Move to the last page of results Click the Page to last icon ( ).
Select the number of items per page In the items per page box, select a number. The minimum number of items per page is 10 and the maximum is 100.
