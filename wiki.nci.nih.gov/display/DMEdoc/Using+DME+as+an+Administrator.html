If your user account has the group administrator or system administrator role, you can manage users, groups, and permissions in DME, and delete data in DME, using the web application (GUI) or Command Line Utilities (CLU).
  • If you want fine-grained control of the activity, or if you want to integrate it into your scientific workflow, consider using CLU.
  • Otherwise, consider using the GUI.
