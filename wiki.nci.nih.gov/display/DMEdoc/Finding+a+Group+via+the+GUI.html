If your user account has the group administrator or system administrator role, you can find a group.
  1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. Click Admin tab > Group. The Manage Group page appears.
  3. In the Group Name field, consider specifying a value. Keep in mind the following points for this field:
    • The system requires exact capitalization. For example, "SMITH" and "smith" do not produce the same results.
    • The system recognizes a percent character (%) as a wild card for any number of characters (zero, one, or more).
    • The system does not recognize any other wild card characters.
  4. Click Search. The search results appear. (If you did not specify any search criteria, the system lists all groups.) For instructions on sorting, filtering, and navigating the search results page, refer to Using Group or User Search Results in the GUI .
