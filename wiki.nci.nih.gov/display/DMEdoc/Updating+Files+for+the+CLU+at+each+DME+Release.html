At each release of DME, perform the following steps:
  1. In a cmd or shell window, type the following command in your working directory to fetch the latest DME files to your local machine: git fetch https://github.com/CBIIT/HPC_DME_APIs The system downloads all changes from the repository.
  2. If you are using Cygwin on Windows and experience a repeating error when running a CLU command after step 1 above, restart Cygwin and run the following command to change end-of-line delimiters from Windows standard to Unix standard in the scripts folder. Use the following command: sed -i "s/\r$//" $HPC_DM_UTILS/scripts/*
