After you select a collection in the Browse page (as described in Browsing for Data via the GUI ), a detail panel for the selected collection appears. The following table describes the actions you can take on the detail panel.
To do this ... Do this ...
Sort the list Click any column header (except Download). To reverse the sort, click it again. To clear the sort, click it a third time.
Filter the list Click the text box in the File Name column header. Type a value. The system displays only the results that contain the specified value.
Download a collection or file In the row for the collection or file of interest, click the download icon ( ). The Download page appears. For details, refer to Downloading Data via the GUI .
