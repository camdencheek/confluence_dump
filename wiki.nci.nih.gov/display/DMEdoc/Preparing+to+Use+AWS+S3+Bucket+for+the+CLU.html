If you need to move a large volume of data, you can do so using an AWS S3 bucket. DME uses AWS for high performance transfer of large files. If you intend to use an S3 bucket with the DME CLU, prepare as follows:
  1. Use a command line editor (such as vi editor) to create a file with the filename credentials that specifies S3 credential information, as follows: [default] aws_access_key_id = <access key> aws_secret_access_key = <secret access key> region = <s3 region>
  2. Consider placing the credentials file in the default location for CLU commands, as follows: ~/.aws/credentials
Now you can download data as described in the following pages:
  • Using dm_download_collection_s3
  • Using dm_download_dataobject_s3
