If your user account has the Read, Write, or Own permission level on any collection or data file in DME, you can create a bookmark on that collection or data file using the web application (GUI) or Command Line Utilities (CLU).
  • If you want fine-grained control of the activity, or if you want to integrate it into your scientific workflow, consider using CLU.
  • Otherwise, consider using the GUI.
