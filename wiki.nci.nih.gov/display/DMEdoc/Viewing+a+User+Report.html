If your user account has the group administrator or system administrator role, you can view a User report.
  1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. Click the Reports tab. The Reports page appears.
  3. From the Report Type list, select User Report. A list of the users in your DOC appears, as well as date fields.
  4. In the Users field, type a value. The system displays only the results that contain the specified value. Select a user.
  5. (Optional) Adjust the date range in the From Date and To Date fields.
  6. Click Generate. The report appears.
The following table describes the data available in the report:
Row Description
Total Archived Data Size (Bytes) The total size of the archived data registered by the specified user:
  • The first number is in bytes.
  • The second number (in parentheses) is in SI units.
Largest Archived Data File Size (Bytes) The size of the largest archived data file registered by the specified user:
  • The first number is in bytes.
  • The second number (in parentheses) is in SI units.
Average Archived file Size (Bytes) The average size of an archived data file registered by the specified user:
  • The first number is in bytes.
  • The second number (in parentheses) is in SI units.
Total Number of Data Objects Archived The total number of archived data files registered by the specified user.
Total Number of Collections The total number of collections registered by the specified user, subdivided by depth in the hierarchy. For example, the user Jane Doe might have registered one collection at the PI_Lab level, six collections at the Project level, and two collections at the Flowcell level.
Files <with specified size> The total number of archived data files registered by the specified user, subdivided by size.
