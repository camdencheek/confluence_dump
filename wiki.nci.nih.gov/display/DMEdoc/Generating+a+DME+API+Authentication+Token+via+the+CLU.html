If you would like to avoid repeatedly typing your password for every command that requires authentication to the DME API server, generate an authentication token. Consider generating this token each time you update your password.
  1. Run the following command: dm_generate_token
  2. When prompted to do so, specify your password. The system displays data transfer statistics. If the system is able to generate your authentication token, it creates or updates the HPC_DME_APIs/utils/tokens/curl-conf file.
