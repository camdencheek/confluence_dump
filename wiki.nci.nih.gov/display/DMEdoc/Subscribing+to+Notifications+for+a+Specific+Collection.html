1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. View the details of the collection as described in Viewing Metadata via the GUI . The Collection page appears.
  3. In the Path field, click the copy icon ( ). The system copies the path for the collection or file to your clipboard.
  4. Click Manage tab > Notifications. The Subscribe Notifications page appears.
  5. In the Collection Updates row, you have various options:
    • If you are currently subscribed to updates in a collection, you can unsubscribe by clearing the check box next to that collection name.
    • If you want to subscribe to a collection, in the Collection Path field, paste the collection path. Click Add. The system adds the specified path to the page.
  6. For each collection path, you have the following options:
    • If you want to receive notifications for file registration in the specified path, but not for any other changes in that path, select Data Registered Updates Only.
    • If you want to receive notifications for all changes in the specified path (collection creation, metadata updates, and file registration), clear the check box.
  7. Click Update. The system updates your profile. The next time an event occurs to which you have subscribed, the system sends you an email notification.
For instructions on selecting notification types, refer to Subscribing to Download and Registration Notifications .
