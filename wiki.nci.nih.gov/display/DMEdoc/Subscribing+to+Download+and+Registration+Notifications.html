Any DME user can subscribe to notifications regarding downloads from (and registrations into) specific collections. If the collection to which you subscribe has child collections, the system sends you notifications for those child collections as well. To subscribe to notifications:
  1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. Click Manage tab > Notifications. The Subscribe Notifications page appears.
  3. For each collection to which you want to subscribe, perform the steps described in Subscribing to Notifications for a Specific Collection .
  4. For each notification type to which you want to subscribe, select the check box in the Subscribe column.
    Sort Notification Type Description
    1 Data Registration Completed The system has successfully transferred the data to the specified location in DME.
    2 Data Registration Failed A failure happened while the system was registering the data.
    3 Data Registration URL Expired A URL for data registration has expired.
    6 Bulk Data Registration Completed The system has successfully transferred the bulk data to the specified location in the data vault.
    7 Bulk Data Registration Failed A failure happened while the system was registering the bulk data.
    4 Data Transfer Download Completed The system has successfully transferred the data from the specified location in DME.
    5 Data Transfer Download Failed A failure happened while the system was downloading the data.
  5. For each notification type to which you want to unsubscribe, clear the check box.
  6. Click Update. The system updates your profile. From now on, each time an event occurs to which you have subscribed, the system sends you an email notification and adds an entry to the My Notifications list on the Dashboard. (For details on a notification, click the link in that list.)
