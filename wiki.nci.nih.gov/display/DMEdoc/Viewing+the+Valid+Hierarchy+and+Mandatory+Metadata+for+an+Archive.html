If your group administrator has given you access to an archive collection, you can view the valid hierarchy and mandatory metadata for that archive.
  1. Log in as described in Logging In via the GUI . The Dashboard appears.
  2. Click theÂ Browse tab. The Browse page appears, displaying content based on access given to you by your group administrator.
  3. In the hierarchy, locate the archive collection you want to understand. Click the information icon next to that collection. A Valid Hierarchy and Mandatory Metadata chart appears. By default, this chart provides the following information:
    • The node names indicate the collection type or types allowed within each collection type.
    • A check mark indicates each collection in which you can register data files.
  4. To view the mandatory metadata for a collection type, click the plus icon next to the node for that collection type. A list of attribute names appear below that node.
  5. To close the Valid Hierarchy and Mandatory Metadata chart, click elsewhere on the page.
