You can export search results from DME, including metadata.To export all search results to a Microsoft Excel (XLS) file:
  1. Perform a search for data, as described in Searching for Data via the GUI . Select the Detailed results option. The search results page appears.
  2. If you want to omit some columns from the export:
    1. In the upper-right corner of the results table, click the menu icon. (DME always exports the UUID column, even if you hide it from the search results.)
    2. Under Columns, click the name of each column you want to omit. An X appears next to that column name. (If you change your mind, click the column name again. A check mark appears next to that column name.)
    3. To hide all columns from search results except Path and actions, select Hide All Columns. (To show all columns, select Show All Columns.)
  3. In the upper-right corner of the page, click Export Search Results. Follow the prompts to save the file.
To export the data from only the current page of search results to a comma-separated values (CSV) file:
  1. Perform a search for data, as described in Searching for Data via the GUI . If you want the export to include metadata, select the Detailed results option. The search results page appears.
  2. Specify the number of records you want to export, by selecting a value from the items per page list.
  3. If you want to export a subset of the rows on the current page: In the selection column, select each collection or data file you want to export.
  4. In the upper-right corner of the results table, click the menu icon.
  5. Select one or more options from the menu, as follows:
    1. If you want to include all columns (except the action columns, Download and Permission) and all rows in the export, select Export all data as csv.
    2. If you want to omit some columns from the export:
      1. Under Columns, click the name of each column you want to omit. An X appears next to that column name. (If you change your mind, click the column name again. A check mark appears next to that column name.)
      2. Select Export visible data as csv.
    3. If you want to export only the selected rows, select Export selected data as csv.
The system exports the data from only the current page of results to a CSV file. For example, if the current page lists 50 records out of 100 total results, the system exports only 50 records.
