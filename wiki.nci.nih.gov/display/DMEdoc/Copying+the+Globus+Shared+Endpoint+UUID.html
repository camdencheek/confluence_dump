DME requires the UUID of your Globus shared endpoint in the following scenarios:
  • If you use the GUI to download data from DME to a Globus endpoint and you choose to specify your endpoint manually
  • If you use the CLU to download data from DME to a Globus endpoint or to register data from that endpoint into DME
Each time you need the UUID, perform the following steps:
  1. Prepare to use Globus, as described in Preparing to Use Globus .
  2. In Globus, navigate to the list of endpoints that are sharable by you. The shared endpoint you created is available in that list with the following description: Shared Endpoint on <name of shareable Globus endpoint>
  3. Click the shared endpoint link. A page appears with details about that endpoint.
  4. Next to the Endpoint UUID field, click the copy icon.
