If you want to preview a list of the source file(s)/folder(s) that the system would register based on what you specified in the Data Source fields, click Dry Run. The system displays a list of the source file(s)/folder(s) that it would have registered based on what you specified in the Data Source fields. If necessary, revise your entries and click Dry Run again until you are satisfied with the dry run list. Keep in mind the following points:
  • This option tests only the Data Source entries. It does not test the Data Destination entries.
  • If you specified a metadata file, specify it again after each dry run.
