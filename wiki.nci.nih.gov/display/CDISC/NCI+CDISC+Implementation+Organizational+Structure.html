The NRDS Organizational Structure will transition into the NCI CDISC Organization Structure.  Note that additional focus groups will be created as content is identified to transition to CDISC compliance.

  • NCI CDISC Implementation Committee

    • Serves as an oversight committee for the NCI CDISC Implementation

  • NRDS CDISC Subgroup

    • Informational subgroup on activities within the NCI CDISC Implementation.  This group will absorbed into the implementation committee or focus groups.

  • CDISC Policy/Governance WG

    • Identify requirements to guide decisions on the adoption and usage of the standardized CDISC CDEs for the NCTN/ETCTN

  • CDISC SAE Integration Focus Group

    • Comprised of SMEs for SAE integration to review the IT impact analysis and determine how to move forward with CDASH/SDTM activities.
    • Review CDISC Implementation Process Workflow
    • Review forms built by CTSU

  • CDISC OPEN/CDUS Focus Group

    • Comprised of SMEs for OPEN/CDUS integration to review the IT impact analysis and determine how to move forward with CDASH/SDTM activities.
    • Review CDISC Implementation Process Workflow
    • Review forms built by CTSU

  • CDISC Validation Focus Group

    • Review tools for CDISC Validation and propose recommendations to the NCI
    • Review CDISC Implementation Process Workflow
