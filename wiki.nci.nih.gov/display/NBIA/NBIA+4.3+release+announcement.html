Dear NBIA Users,   We are pleased to announce the release of version 4.3 of the National Biomedical Imaging Archive (NBIA), available at http://imaging.nci.nih.gov .Some of the new features included in this release are:
  • A Download Manager, a new Java WebStart application that downloads images to your local drive in an unzipped format.   It does not limit the download data size, and also allows users to pause and resume downloads at any time.
  • A Dynamic Search capability, which allows users to construct queries against additional DICOM tags related to collections, patients, studies, image series, and images.
  • A Verify Submission Tool, an administration tool which allows site administrators to generate multiple historical reports, including reports on image submissions for a certain date range.
  • Enhancements to the GUI installer, which allows users to install different components (Web App, Grid service, and CTP) on different physical systems.
Download detailed release notes . The features and screenshots of enhancements planned for the next release (4.4) will be demonstrated at the next user group meeting, currently scheduled for the end of October 2009.For groups interested in installing NBIA 4.3 locally, the full installation package as well as an upgrade installer and associated documentation are available on SVN .As always, support is available from Application Support .
