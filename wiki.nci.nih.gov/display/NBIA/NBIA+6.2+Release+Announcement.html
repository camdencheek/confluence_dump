The NBIA team is pleased to announce the release of version 6.2 of the National Biomedical Imaging Archive (NBIA) . This release updates the infrastructure to conform to the new NCI environment, upgrades CSM to 5.2, provides improvements to search, an important bug fix to Oviyam, as well as additional bug fixes and minor enhancements.

Infrastructure Update

NBIA infrastructure has been updated to allow for continued support of the system by moving to recent releases as well as the movement of the NBIA application to NCI's Shady Grove facility. This includes:
  • Java 7
  • JBoss 6.2 EAP
  • Oracle 11

CSM 5.2

CSM has been updated to version 5.2 with additional encrypted fields for better security and updated infrastructure

Search Improvements

All searches now include guidance on performing searches, and simple search now includes the ability to search on date available as well as subject IDs.

Oviyam Update

The release includes an update to the integration of the embedded Oviyam 2 application that allows it to work correctly across multiple studies.
