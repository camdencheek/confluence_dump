Dear NBIA Users,   We are pleased to announce patch release 4.2.1 of the National Biomedical Imaging Archive (NBIA) software available at http://imaging.nci.nih.gov .This release includes an additional NBIA grid service operation (getRepresentativeimageBySeries) and also adds an authentication mechanism (to address a security vulnerability in the submission software)  to submit data to the NCI CBIIT instance.Download detailed release notes .For groups interested in installing NBIA 4.2.1 locally, the full installation package as well as an upgrade installer and associated documentation was available at the time of release.As always, support is available from Application Support .
