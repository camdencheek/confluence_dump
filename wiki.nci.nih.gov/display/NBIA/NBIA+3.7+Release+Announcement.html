Dear NCIA Registered Users,   We are happy to announce release of NCIA version 3.7 at http://imaging.nci.nih.gov This release includes a new quick-preview "movie" viewer feature and bug fixes. This new feature is summarized below. As always, support is available from Application Support .It's now possible to preview a complete image series in a movie viewer in both automatic and manual mode.•    Manually advance slices forward and backward (ex. Sliding bar) •    Automatically advance slices forward and backward •    point and click to any slice (Click on any point of a sliding bar) •    loop through slices (continue to loop forward and then backward until user termination) •    mark first and last slice boundaries for viewing •    scroll to first, last, next, and previous image
