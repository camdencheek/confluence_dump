Reviewing Saved Query Information

To review a saved query
  1. Select Admin > Manage Saved Queries.The Manage Saved Queries page appears listing all the queries saved by NBIA users.
  2. The following information is listed for each query.Sorting the tableBy default, the table is sorted by Query Name, but you can also sort by the Last Executed or Query Creator columns.
    Column Name Description
    Query Name Click the name of the query to display the Criteria Selected by the user. The selected field name(s) and data are listed.
    Last Executed Lists the date and time that the query was run.
    New Data Available indicates that new data is available.
    Query Creator Username of the person who saved the query.

Deleting a Saved Query

To delete a saved query
  1. Select Admin > Manage Saved Queries.The Manage Saved Queries page appears.
  2. Check the box next to the Query Creator column of a query to be removed. You can delete multiple queries at once.Removing All Saved QueriesTo remove all of the saved queries, check the box next to column name, Query Creator. All queries are selected.
  3. Click Remove Selected Items.You are prompted to confirm the deletion and the selected queries are removed.
