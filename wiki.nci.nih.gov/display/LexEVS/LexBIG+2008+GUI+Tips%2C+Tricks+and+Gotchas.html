This page is intended to provide a ready-reference for the LexBIG Graphical User Interface, covering materials for inclusion in the next release of the LexBIG Programmer and Administration Guides.As primary reference, refer to the sections on the GUI in each of these guides downloadable from the files tab of the LexBIG GForge project and included as part of installed materials in the /doc folder. Link to files tab provided for historical purposes If you need access to these files, please contact Application Support: ncicbiit@mail.nih.gov https://gforge.nci.nih.gov/frs/?group_id=14
  • Administrator's Guide - Information on the upper portion of the GUI (used primarily to load and maintain terminologies) is included in the Administrator's Guide.
  • Programmer's Guide - Information on the lower portion of the GUI (used primarily to test and exercise the API) is included in the Programmer's guide.
