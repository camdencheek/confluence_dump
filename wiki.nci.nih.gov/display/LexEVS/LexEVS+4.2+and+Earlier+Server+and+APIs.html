LexEVS Grid Service v4.2

The LexEVS Grid Service provides another programmable interface that can be used to obtain access to data served by LexEVS 4.2.
  • Access
    • URL: Link provided for historical purposes http://lexevsapi.nci.nih.gov/wsrf/services/cagrid/LexEVSGridService
    • caGrid Portal
  • Documentation updates are limited since this is a point release. There is a User Guide posted on GForge for the LexEVS 4.2 Grid Service to provide support for programming to this interface.
  • Sample Test Client - This Sample Test Client Zip file on GForge contains sample source code and the required libraries needed to use the LexEVS Grid Service. The download may take some time.

LexEVS Grid Service Previous Releases

The EVS Grid Service 1.0 (pointing to the caCORE 3.1 EVS Services) is still available. It will be decommissioned with next major release of the LexEVS (v5.0). Therefore, the EVS Team encourages migration to the current release of the LexEVS Grid Service (v4.2).As of the 4.1 Release of the EVS Grid Service, the service is no longer part of the caGrid infrastructure and has been deployed as a separate unit.To access the EVS 4.1 Grid Service, visit the EVS Archives . Note that the 4.x license terms apply.
