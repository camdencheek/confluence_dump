Summary

The goal of this project (S13-500 MOD4, MOD5, MOD6, MOD7, MOD8) is to address JIRA items and provide maintenance and support for LexEVS 6.2 and to develop and release both LexEVS 6.3 and LexEVS 6.4 (6.4.1).Additional documentation for the project will be added throughout the lifecycle of the project.

Document Purpose

The purpose of this document is to satisfy the project documents requirement to support project S13-500 MOD4 for the National Cancer Institute Center for Biomedical Informatics and Information Technology (NCI CBIIT) for LexEVS Release 6.3 and LexEVS Release 6.4 (6.4.1).
