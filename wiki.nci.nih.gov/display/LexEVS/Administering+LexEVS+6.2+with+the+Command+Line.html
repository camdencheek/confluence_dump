LexEVS 6.2 has all the administrative capabilities of LexEVS 6.0 and LexEVS 6.1 as well as the following tagging functions:

Tagging a Vocabulary

Shell Script Use and Function
RemoveTagScheme  Removes tag scheme. Options:
  •  u,-- urn <name> URN uniquely identifying the code system.
  • -v,-- version <id>Version identifier.
Remove Tag Example: RemoveTagScheme -u "urn:oid:2.16.840.1.113883.3.26.1.1" -v "05.09e"
