Supported Platforms

The servers listed in the table below represent hardware that LexEVS has been tested on.WarningThese servers do not represent the minimum hardware required for loading any terminology. The LexEVS Local Runtime performs adequately in an environment with a memory limit of 4GB, but users cannot load larger OWL files, such as the full NCI Thesaurus or NCI Metathesaurus, in such environments. Users will have to load such terminologies on more powerful hardware. There does not exist a list showing the minimum hardware for a given terminology.
