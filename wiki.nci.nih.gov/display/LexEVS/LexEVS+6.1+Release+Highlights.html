LexEVS 6.1 was released on May 6, 2014.Full details are available on the LexEVS 6.1 page.LexEVS 6.1 features include:
  • New REST interface compliant with the OMG Common Terminology Service - Release 2 (CTS2) 1.0 specification .
    • Quick start guide to using the new CTS2 API , with sample code.
    • REST support for bulk download of terminology data.
    • New terminology loaders supporting  OWL2, native MedDRA, HL7 MIF, and resolved value set data.
    • Optimized “Contains” search, as seen in the 2.3 and 2.4 NCI Term Browser.
    • Value set performance improved.
    • Backwards compatibility with LexEVS 6.0.
    • Note on 6.0 Grid Services: These are not supported in 6.1, and are planned for retirement by October 2014.
  • The Release Notes give more information on fixes, enhancements, and known issues.
 LexEVS 5.1 is being retired. All users should migrate to LexEVS 6.
