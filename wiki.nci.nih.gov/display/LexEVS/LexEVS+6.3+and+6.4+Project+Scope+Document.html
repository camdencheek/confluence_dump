Project Scope

The purpose of this document is to collect, analyze, and define high-level needs and features of the National Cancer Institute Center for LexEVS Release 6.3 and LexEVS Release 6.4. This document focuses on the functions proposed by the product stakeholders and target users in order to make it a better product.6.3 will not be released as a production release.  It will be used as an internal development release for use by the NCI team only.  All features and fixes from 6.3 will be officially released as 6.4.

Vision and Needs

The objective of this release of the LexEVS is to provide a bug fixes and a set of new features as captured in the document.
