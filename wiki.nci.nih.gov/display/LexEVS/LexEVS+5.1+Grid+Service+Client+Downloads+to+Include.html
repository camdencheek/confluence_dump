LexEVS Analytical Services Client Enables Java programs to establish a connection to the LexEVS caGrid Analytical Services. LexEVS_51_caGRIDServices_analytical_client.jar LexEVS Analytical Services Client Dependencies (Minimal) Includes only code required by the grid services, packaged as a ZIP file for convenient deployment. This allows the flexibility to provide your own 3rd party dependencies. It does not include other 3rd party dependencies. These dependencies can be used in a caGrid 1.2 or 1.3 environment. NOTE: With this package a user would be responsible for providing all dependencies.
