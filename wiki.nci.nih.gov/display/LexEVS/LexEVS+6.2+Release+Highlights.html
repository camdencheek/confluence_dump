LexEVS 6.2 was released in October 2014.Full details are available on the LexEVS 6.2 page.LexEVS 6.2 features include:

Updated CTS2 Service to conform to CTS2 1.1

This release of LexEVS focuses updating the CTS2 compliant REST service from the 1.0 to 1.1 version of the CTS2 specification. This updated CTS2 compliant service provides core support of the Common Terminology Services 2 (CTS2) 1.1 functionality as described in the OMG CTS2 specification 1.1 .

Additional Functionality

Also included in this package:
  • Software component upgrades
  • Implementation of a prototype value set viewer
  • Bug fixes
  • The Release Notes give more information on fixes, enhancements, and known issues.
