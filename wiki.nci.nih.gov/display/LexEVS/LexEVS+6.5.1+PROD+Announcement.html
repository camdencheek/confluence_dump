Three new properties will be added to the NCI Thesaurus(NCIt) as part of this release.
  • One is a data property and the other two are internal administration properties.
  • Maps_To - A data property that indicates a term in an external terminology maps to the annotated concept.
  • The annotations (qualifiers) specify which terminology and identifier is the target of the mapping.
Example: C116374 Ageusia    Maps_To  Ageusia            Target_Terminology: MedDRA            Relationship_to_Target: Has Synonym            Target_Term_Type: LLT            Target_Code: 10001480Publish Value Set - An internal administration property that tags a value set as curated and ready for publicationTerm Browser Value Set Description - An internal administration property that provides descriptive text of a value set for use in our Term browser. Please feel free to contact EVS staff at NCIThesaurus@mail.nih.gov   for additional information and to provide feedback as necessary.
