Managing Graph Node Resolution Database Loads

The graphing database ArangoDb must be available to LexEVS (See the system requirements here ).  It's configuration must also be created in the lbconfig.props.  Loading of graphs for multiple terminology versions is not available at this time.  Setting STRICT_ARANGO_REQUIREMENT=false allows LexEVS to start up without an ArangoDb configuration or installation.  Refer to ArangoDb documentation for setting any other variables.Sample lbconfig props configuration GRAPH_DB_USER=root GRAPH_DB_PWD=lexgrid GRAPH_DB_URL=localhost GRAPH_DB_PORT=8529 GRAPH_DB_MAX_CONNECTIONS=200 GRAPH_DB_CONNECTION_TIMEOUT_LENGTH=60000 STRICT_ARANGO_REQUIREMENT=false

Load of LexEVS Graph Nodes To ArangoDb
