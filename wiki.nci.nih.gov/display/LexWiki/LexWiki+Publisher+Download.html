There are two downloads for LexWiki Publisher depending on your LexEVS install. If you already have a successful LexEVS 6.0 installation, then you will already have access to the library file named "lbRuntime.jar" and do not need to download it. These downloads do not include that file and therefore are faster downloads:
  • LexWikiPublisher 2.0 (ZIP) (3 MB) or LexWikiPublisher 2.0 (TAR) (4 MB)
  • LexWiki Publisher 2.0 configuration file schema
The download for the previous release is also available: LexWikiPublisher 1.0 zip (33 MB)
