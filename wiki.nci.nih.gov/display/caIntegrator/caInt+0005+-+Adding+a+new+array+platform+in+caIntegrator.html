Question: How do I add a new array platform in caIntegrator?

Topic: caIntegrator Usage Release: caIntegrator 1.2 and above Date entered: 03/23/2011

Answer

You can create a study without genomic data, but you cannot add genomic data to a caIntegrator study without a corresponding supported array platform. Currently, caIntegrator only supports following types of platforms: Affymetrix Gene Expression, Affymetrix Copy Number, Agilent Gene Expression, and Agilent Copy Number. Please refer to the User Guide for step-by-step instructions on how to create a new platform. To add a new platform, the array annotation file is required. For Affymetrix platforms, please refer to GeWorkbench025 for how to obtain the annotation file. For other platforms, please refer to the following resources:
  • caArray
  • GEO
  • Arrayexpress
