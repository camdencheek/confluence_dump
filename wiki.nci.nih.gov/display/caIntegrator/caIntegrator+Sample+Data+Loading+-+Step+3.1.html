The page "Edit Genomic Data Source" opens. Fill the "Data Source" form by filling in or selecting values as shown and click Save. Description of "Edit Genomic Data Source" Form The fields to complete are Web URL ( https://array.nci.nih.gov/caarray in the example), Server Hostname (array.nci.nih.gov in the example), Server JNDI Port (here 8080), Experiment ID (here jacob-00182), select Vendor (here Affymetrix), select Data Type (here Expression), select Platform (here HG-U133a), select Central Tendency for Technical Replicates (Mean in the example), and select Standard Deviation Type (Relative (Percentage) in the example). Also check the box for "Technical Replicates have statistical variability" if applicable. Finally, click the "Save" button at the bottom.
