Problem: When I install caIntegrator using the GUI wizard and try to access the application using the URL provided in step 29 of the wizard, I get a 404 'Page Not Found' error message in my browser.

Topic: caIntegrator Installation and UpgradeRelease: caIntegrator 1.3 and aboveDate entered: 01/27/2012

Solution

The URL listed in step 29 of the installation wizard (highlighted in the screenshot below) is incorrect: The correct URL is as follows: http://localhost:47210/caintegrator/ The host name of the machine caIntegrator is installed on can be used in place of the phrase 'localhost' in the above URL. In the above screenshot, the URL contains the host name 'MAGNET14' in place of the phrase 'localhost'.
