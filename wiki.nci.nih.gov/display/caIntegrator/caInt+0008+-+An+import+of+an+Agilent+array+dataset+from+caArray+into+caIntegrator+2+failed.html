Question: An import of an Agilent array dataset from caArray into caIntegrator 2 failed. Why?

Topic: caIntegrator Usage Release: caIntegrator 1.2 and above Date entered: 08/31/2010

Answer

For Agilent array data, the probe names have to start with "A_". This lets the code ignore all of the other irrelevant data in a real Agilent design file.
