This caIntegrator v.1.4 or 1.4.x Installation Guide provides you with the instructions to install and configure a fresh caIntegrator v.1.4 or 1.4.x application. The caIntegrator installation installs and configures a JBoss application server and creates a caIntegrator-specific schema within a pre-existing database on a preinstalled MySQL server.Directions are given in this document for both Linux and Windows operating systems.
