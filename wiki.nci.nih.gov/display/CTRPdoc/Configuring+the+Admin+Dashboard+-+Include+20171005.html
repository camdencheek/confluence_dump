You must have a username and password to use the web application. Additionally, you must have special permission to run the reports. For assistance, contact us at ctrp_support@nih.gov .
  1. Visit https://trials.nci.nih.gov/strap . The login page appears. Specify your user name (email address) and password. Click Sign In. Follow the prompts to complete 2-step verification. The Admin Dashboard appears, with information about trials.
  2. To view information about accruals, click Accrual.
  3. To configure all of the charts globally, click Dashboard Settings. The global settings panel appears. All fields are optional. The following table provides instructions for each field:
    Field Instructions
    Cancer Centers Select a Cancer Center (CTRP organization family).
    Disease Specify a disease name.
    Trial Status Select one or more trial statuses. For information, refer to Trial Status Values in the CTRP and ClinicalTrials.gov .
    Study Source Select one or more study sources. For information, refer to CTRP Trial Categories, Study Sources .
    Trial Type Specify a trial type.Â For information, refer to Trial Types and Subtypes .
  4. For each chart, various options are available in a side bar. To expose the side bar, hover over the right edge of the chart:
    • To expand the chart, click the Expand icon.
    • To specify a different disease for the chart, click the Settings icon. The Chart Settings panel appears. In the Disease field, specify a disease name. Click OK.
    • To hide or show specific values in the chart, click the Hide/Show icon. Click each value that you want to hide or show.
    • To download the chart data, click the Download icon.
    • To capture an image of the chart, click the Screenshot icon. The Screenshot Chart panel appears. You can specify a different chart title, file name, range of colors, and image quality. Click Download.
    • To lock the chart from global changes, click the Lock icon.
