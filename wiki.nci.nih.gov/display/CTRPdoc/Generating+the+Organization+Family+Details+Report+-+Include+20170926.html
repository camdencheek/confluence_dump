You must have a username and password to use the web application. Additionally, you must have special permission to run the reports. For assistance, contact us at ctrp_support@nih.gov .
  1. Visit https://strap.trials.nci.nih.gov/login . The login page appears. Specify your user name (email address) and password. Click Sign In. Follow the prompts to complete 2-step verification. The home page appears. Click Automated Reports. The Report Settings dialog box appears for the CTRP Data Table 4 report.
  2. Click Cancel. From the list of reports, select Organization Family Details. The Report Settings dialog box appears for that report.
  3. Specify the Cancer Center organization families for which you want to run a report:
    1. To run a report for specific organization families, select Select Multiple. In the list of Cancer Centers, select one or more organization families.
    2. To run a report for all organization families, select Select All.
  4. In the Date Range fields, specify a range of time. For each date field, click the calendar icon to select a date.
  5. Click Submit. The system loads the report. For information about the columns included in the report, refer to Data Elements Included in Organization Family Details Report .
To return to the Report Settings dialog box, click Report Settings. For additional instructions, refer to Working with CTRP Automated Reports .
