This section describes how to record, update, and delete the total number of accruals (accrual count) associated with Abbreviated and Other clinical trials, via the web application. For instructions for submitting accrual data for Complete trials, refer to Working with Complete Trial Accruals . For instructions on submitting accrual data via batch upload, refer to Submitting Accrual Data Batch Files .Pharmaceutical companies are not users of CTRP, therefore each participating site on an Abbreviatedtrial must enter its own accrual count into CTRP.
