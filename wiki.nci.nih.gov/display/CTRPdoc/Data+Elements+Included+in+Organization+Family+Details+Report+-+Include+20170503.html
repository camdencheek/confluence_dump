The following table describes the data elements displayed in the Organization Family Details report for each trial:
Data Element Description
Family Name Name of the CTRP organization family (Cancer Center).
PO ID ID that NCI assigned to the organization.
CTEP ID ID that the Cancer Therapy Evaluation Program (CTEP) assigned to the organization.
Organization Name Name of the organization. If an organization is a member of the specified organization family on any date in the reporting period (on or after the reporting period start date and on or before the reporting period end date), then the system includes this organization in the report.
Status The curation status for the organization record. For details, refer to Changing the Curation Status .
Relationship Functional relationship of the organization to the family.
  • Organizational – A formalized grouping of organizations.
  • Affiliation – Organizations that share common working agreements or business relationships. For example, an independent hospital or clinic working with a university or institution is an affiliate of the university or institution.
Start Date Date on which the organization's relationship with the family began.
End Date Date on which the organization's relationship with the family ended.
Address 1 First address line for the organization.
Address 2 Second address line for the organization.
City City in which the organization is located.
State State or province in which the organization is located.
Postal Code Postal code in which the organization is located.
Country Country in which the organization is located.
Email Email address for the organization.
Phone Phone number for the organization, including any extension.
