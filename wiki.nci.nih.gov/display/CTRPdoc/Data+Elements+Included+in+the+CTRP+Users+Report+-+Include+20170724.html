The system populates many of the data elements in the CTRP Users report from information provided by users about themselves in the CTRP Registration web application, when they create and manage CTRP accounts. The system populates other data elements (such as roles) from other CTRP modules (such as the PA and PO web applications).This page describes each data element and provides links to other pages that indicate how users add or view the data in each CTRP user account. The parenthetical with each link indicates the relevant CTRP web application for that link. For general account information, refer to the following pages:
  • Creating a CTRP Account (Registration)
  • Managing Your Account (Registration)
  • Accessing Registered User Details (PA)
The following table describes the data elements displayed in the CTRP Users report:
Data Element Description
First Name First name of the CTRP user.
Last Name Last name of the CTRP user.
Email Address Email address of the CTRP user.
Phone Phone number of the CTRP user.
Status Status of the CTRP user, as set by the CTRP Support team. If you have questions about the status of a CTRP user, contact ctrp_support@nih.gov .
Activation Date The date on which the CTRP Support team activated the CTRP user.
Inactivation Date The date on which the CTRP Support team inactivated the CTRP user (if the user is currently inactive).
Roles All roles (user groups) for the CTRP user. For information, refer to the following pages:
  • Managing User Groups in PA (PA)
  • Managing User Groups (PO)
Is Site Admin? Value indicating whether the CTRP user is a site administrator. For information, refer to the following pages:
  • Requesting Organization Admin Rights (Registration)
  • Granting and Revoking Administrative Rights (Registration)
  • Processing Requests for Admin Rights (PA)
  • Managing Site Administrators (PA)
PRS Organization Name The ClinicalTrials.gov Protocol Registration System (PRS) organization specified in the CTRP user account profile.
Login Name The login name of the CTRP user.
Affiliated Organization Family Name Name of the organization family in which the CTRP user's affiliated organization is a member.
Affiliated Organization PO ID The ID that NCI assigned to the CTRP user's affiliated organization.
Affiliated Organization CTEP ID The ID that CTEP assigned to the CTRP user's affiliated organization.
