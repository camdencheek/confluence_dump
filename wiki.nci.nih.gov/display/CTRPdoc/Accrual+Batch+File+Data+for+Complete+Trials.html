The CTRP Accrual Batch File conforms to the CDUS file specification, but it only uses a limited number of fields from that file. Data in the rest of the fields are ignored, but the place holders (commas) must still be present in the data file, whether empty or filled. Before you beginCTRP considers each accrual submitted as the total (cumulative) count to date.
