How to Abstract Trial Status History Information
  1. Search for the trial of interest. In the search results, click the NCI Trial Identifier link for that trial.
  2. On the Administrative Data menu, click Trial Status. The Trial Status page appears.
  3. On the upper right corner of the Trial Status page, click History.If all transitions are valid, the Trial Status History window displays all trial status records to date, included trial status records previously deleted. The Actions column displays applicable icons.
  4. To add a new interim trial status, click Add New Status and follow the instructions in Abstracting Trial Statuses .The Add Trial Status dialog box appears.
  5. To update a status record, click the Edit icon. The Edit Trial Status window displays current trial status information.
  6. Click Validate Status Transitions.
  7. Make the change as necessary, enter a comment, and then click Save. If the system invalidates the new record, correct the errors and save the record.
  8. To delete a status record, click the Delete icon, and then click Save.You must enter a comment any time you change or delete a trial status record.
  9. To display "Why Study Stopped" comments, click the View Comment icon.
  10. To display historical trial status audit information, click the View Audit Trial icon.The Audit Trial window displays current and historical status records for the trial.
  11. Click the X in the upper right corner to return to the Trial Status History window.
