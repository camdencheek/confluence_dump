There are three main areas of organization detail information: basic identifying information, address information, and contact information.For more information about organization updates in CTRP, refer to the following: How to Modify Organization Details
  1. Navigate to the record of interest.
  2. In the Action column for a given organization, click Curate. The Organization Details page displays all information about the organization provided to date.
  3. If you are not the record owner, click Override.
  4. Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Items with an asterisk are required fields. If you need to change the curation status, see Changing the Curation Status before you continue to the next step.
    Field Label Description/Instructions
    Status* Select the current curation status. For status descriptions, see Changing the Curation Status .At this level, you can change the curation status for organizations with CTEP IDs. However, it would be better to perform a CTEP Import instead. (At the role level, you can not change the curation status for organizations with CTEP IDs.)
    Organization Name* Enter the official name of the organization.The following organization names must remain as they are. Do not attempt to change these organization names, because these organizations act as Identifier Assigners on trials, so the system refers to them by their exact names:
    Org ID Organization Name
    16108126 NCI - Center for Cancer Research
    502 Cancer Therapy Evaluation Program
    3830539 ClinicalTrials.gov
    66707 National Cancer Institute Division of Cancer Prevention
    154376 National Cancer Institute
    Veterans Administration Specify whether the organization is a Veterans Administration organization.
    Aliases Enter alternate names for the organization. Click Add after you enter each alternate name.
    Address Information* section Provide as much information about the organization as possible. All fields, except Address Line 2, are required. If the organization is located in a country that does not use postal codes, type NULL in the Postal Code field.

    Comments Enter any information that you want associated with the organization record. See Adding and Viewing Comments for detailed instructions.
  5. Click Save.
