The Inbox lists Registration users who have requested administrative privileges for their organizations.Each organization registered in the CTRP system can have one primary site administrator (Admin) who is responsible for reviewing and updating user access to trials owned by the organization, and for assigning site administrator status to other users in the organization.The Inbox lists Admin access requests by default. You can accept or reject these requests on a per-user basis.How to Grant Admin Rights to Registration Users
  1. On the top section of the main menu, click Inbox.The Pending Admin User page appears. Requests for Site Administration, if any, are listed by default. The time stamp is recorded when the user requests Admin rights in Registration.
  2. In the First Name column, click the link for the person to whom you want to accept or reject a request.The requesterâ€™s name and affiliation information is displayed.
  3. Do one of the following:
    • To accept the request, click Accept. -or-
    • To reject the request, click Reject.
The request is removed from the list of pending Admins, and the user is notified about the acceptance or rejection via email.
