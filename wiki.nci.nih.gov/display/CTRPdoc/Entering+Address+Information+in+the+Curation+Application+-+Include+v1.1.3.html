When adding or editing role information for persons and organizations, you can use an existing entity’s postal address rather than entering the information anew.How to Enter Address Information
  1. In the Address Information section, click Add Postal Address. The Add Postal Address page appears. Postal Address Page-Organizations Postal Address Page-Persons
  2. Do one of the following to add the postal information:
    • If the entity has the same address as the one you are curating, click Use Organization’s/Person’s Address to populate the address fields automatically. - or -
    • From the Country drop-down list, select the country where the person/organization receives its mail, and then complete the rest of the address fields. Provide as much information about the person/organization as possible. Then click Save. Fields marked with an asterisk are required. If the person/organization is located in a country that does not use postal codes, type NULL in the Postal Code field.
