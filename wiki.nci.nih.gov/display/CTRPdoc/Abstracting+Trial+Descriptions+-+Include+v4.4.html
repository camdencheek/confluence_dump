Trial descriptions include the following information:
  • Brief title
  • Brief summary
Additionally, they may include a detailed description.How to Abstract Trial Descriptions
  1. Search for the trial of interest. For instructions, refer to Searching for Trials in PA .
  2. In the search results, click the NCI Trial Identifier link for that trial. The Trial Identification page appears.
  3. On the Trial Identification page, check out the trial. For instructions, refer to Checking In and Checking Out Trials . (This checkout step is optional for Super Abstractors.)
  4. On the Scientific Data menu, click Trial Description. The Trial Description page appears.
  5. On Trial Description page, in the various fields, specify the appropriate information. The following table describes the fields. An asterisk (*) indicates a required field. You can expand text fields and table cells to reveal all of their contentTo increase the size of a text field or table cell, click and drag the lower right corner handle.
    Trial Description Instruction
    Brief Title* Enter a title that summarizes the purpose of the trial.
    Brief Summary* Enter a summary description of the trial.
    Detailed Description Optionally, enter all the information that accurately describes the trial in detail.
  6. Click Save.
