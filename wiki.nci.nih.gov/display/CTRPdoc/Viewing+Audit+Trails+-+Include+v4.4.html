The Protocol Abstraction System records changes that all abstractors make while processing original, amended, and updated trials.The Audit Trail feature is available for changes to Marker and NCI-Specific information only. How to View Audit Trails
  1. Search for the trial of interest. For instructions, refer to Searching for Trials in PA .
  2. In the search results, click the NCI Trial Identifier link for that trial. The Trial Identification page appears.
  3. On the Trial Overview menu, click Trial History, and then click the Audit Trail tab. The Audit Trail page appears.
  4. In the Audited Object field, select either NCI-Specific Information or Markers.
  5. In the Start and End Date fields, enter the start and end dates (time range) over which you want to view changes to the trial.
  6. Click View Audit Trial. The Audit Trail page displays the audit trail records you requested.
  7. To export the audit trail, at the bottom of the page, click either CSV (comma-separated values), Excel, or XML (extensible markup language) as your target format.
