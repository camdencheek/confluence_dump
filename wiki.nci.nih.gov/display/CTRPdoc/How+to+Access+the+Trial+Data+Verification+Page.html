The Update/Verify Trial Data page can be accessed by either of the following methods below:
  • Using the Trials to Verify menu
  • Using the Search menu
The Trials to Verify menu displays a count of trials assigned to the user requiring verification, this value will dynamically update once trials are verified. Using the Trials to Verify menu:
  1. Select the Trials to Verify menu from the main toolbar. The Trials Needing Verification page displays.
  2. Search for the trial in question by using the Search: box to perform a keyword search.
  3. Once the trial has been identified, perform one of the following actions:
    1. Click on the Identifier in the NCI Trial Identifier column:
    2. Scroll horizontally to the Available Actions column and click Select Action > Update/Verify.
    3. The Update/Verify Trial Data page appears.
    4. For information on updating and verifying trials, see the following: How to Verify Trial Data
Using the Search menu:
  1. From the Search menu, select Clinical Trials. The Search Clinical Trials page displays.
  2. Search for the desired trial(s) by using any of the filter options available on the page, or select the Search My Trials option from the Search button.
  3. On the Clinical Trials Search Results page, scroll horizontally to the Available Actions column and click Select Action > Update/Verify.
  4. The Update/Verify Trial Data page displays.
  5. For information on updating and verifying trials, see the following: How to Update/Verify Trial Data
