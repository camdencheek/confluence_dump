For trials that use ICD-O-3 disease coding, select a site to associate with a study subject. How to Search for Sites
  1. Select the trial you want to work with by following instructions in Searching for and Selecting Your Trials , and clicking the corresponding NCI Trial Identifier link.
  2. At the bottom of the Search Study Subject section, click Add New Study Subject. The Add Study Subject page appears.
  3. To assign a site to the study subject, next to the Site field, click Look Up. The Search Sites window appears.
  4. In the Site Name field, type part or all of the site being studied. The Site Code System, ICD-O-3 (International Classification of Diseases for Oncology, 3rd Edition) is preselected for you.Search tipsType as many letters of the site name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the series of letters you type) for you implicitly. You can type wildcard symbols (%) between characters of the string as necessary.
  5. Press Enter, or click Search. The system searches for the site in the database and returns a list of results.
  6. Navigate to the site being studied, and click Select. Refer to Working with Accrual Tables and Search Results . The system populates the Site field for you.
