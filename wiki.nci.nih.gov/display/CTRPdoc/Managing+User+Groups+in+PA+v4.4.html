This page provides instructions for security administrators. A security administrator is a user with administrative privileges who is responsible for adding users in the organization to groups and removing users from groups.How to Manage User Groups
  1. On the main menu, click Manage User Groups. The Manage User Groups page appears.
  2. If necessary, adjust the display of users:
    • To display the next (or previous) page of users, click Next (or Previous).
    • To display more users on each page, click the Show entries list and select another option.
    • To filter the list of users, in the Search field, specify all or part of an LDAP ID.
    • To sort the list of users or groups, click the triangle in one of the column headers.
  3. Adjust the assignment of users to groups:
    • To add a user to a group, click in the Groups field for a user and select a group.
    • To remove a user from a group, click the x for that group in the row for that user.
The following table describes the available groups:
Group Description
Submitter Users in this group can log into the CTRP Registration application.
RegAdmin Users in this group can log into the CTRP Registration application and access the Batch Upload option in the Register Trial menu.Deprecation of Trial Registration Batch UploadWe have removed all users from the RegAdmin group as part of the deprecation of the Trial Registration Batch Upload feature. RegAdmin still exists as a security group/role, but it will no longer be assigned to users unless specifically requested by CTRP leadership.
ReportViewer Users in this group can log into the CTRP PA application and for read-only access. This group should be provided to only internal NIH staff and team.
SuAbstractor Users in this group can log into the PA application with the Super Abstractor role.
gridClient Users in this group can call PA Grid/REST services.
AdminAbstractor Users in this group can log into the PA application with the Administrative Abstractor role.
ScientificAbstractor Users in this group can log into the PA application with the Scientific Abstractor role.
ResultsAbstractor Users in this group can log into the PA application with the Results Abstractor role.
SecurityAdmin Users in this group can log into the PA application and manage user groups.
ProgramCodeAdministrator Users in this group can log into the Registration application and manage program codes for any organization family (Cancer Center family of organizations). To access the Program Codes option on the Administration menu, the user account must also be affiliated with an organization family in CTRP.
Combining any of these groups for a user results in combined permissions for that user. For example, a user in the AdminAbstractor and ScientificAbstractor groups can log into the PA application with the Scientific/Administrative Abstractor role. For details, refer to Role-based Tasks in PA .
