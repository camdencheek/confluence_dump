The system evaluates participating site statuses on a trial to determine whether that trial should be included in a Data Table 4 report. The following table demonstrates whether the system includes trials in the Data Table 4 report or excludes trials from the Data Table 4 report, based on participating site status:
Example Before the Reporting Period During the Reporting Period After the Reporting Period Result
Example 1 (Not relevant to the example.) The participating site status is open, as defined in Types of Trials Included in the CTRP Data Table 4 Report . The participating site status is Withdrawn. The trial is included in the Data Table 4 report.
Example 2 (Not relevant to the example.) The participating site status is open and then Withdrawn. (Not relevant to the example.) The trial is included in the Data Table 4 report.
Example 3 The participating site has a status of Withdrawn. (Not relevant to the example.) (Not relevant to the example.) The trial is excluded from the Data Table 4 report.
After the system determines that a trial should be included in a Data Table 4 report, the system includes all accrual for the organization family on that report.
