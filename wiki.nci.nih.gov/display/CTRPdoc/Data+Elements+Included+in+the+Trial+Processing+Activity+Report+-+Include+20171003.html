The following table describes the data elements displayed in the Trial Processing Activity report:
Data Element Description
Submitted Trials Number of trials submitted in the specified time period.
Abbreviated Trial Category Submissions Number of Abbreviated trials submitted in the specified time period.
Completed Trial Category Submissions Number of Complete trials submitted in the specified time period.
Accepted Trials Number of trials CTRO accepted in the specified time period.
Rejected Trials Number of trials CTRO rejected in the specified time period.
Trials Went On Hold Number of trials CTRO placed on hold in the specified time period.
Still on Hold Status Trials Number of trials CTRO placed on hold in the specified time period, that are still on hold at the time of report generation.
Total Abstracted Trials Number of trials CTRO abstracted in the specified time period.
Abstracted Abbreviated Trials Number of Abbreviated trials CTRO abstracted in the specified time period.
Abstracted Completed Trials Number of Complete trials CTRO abstracted in the specified time period.
TSR Sent Trials Number of trials for which TSRs have been sent in the specified time period.
Amendments Total Trials Submitted Number of trial amendments submitted in the specified time period.
Amendments Accepted Trials Submitted Number of trial amendments CTRO accepted in the specified time period.
Rejected Amendment Trials Number of trial amendments CTRO rejected in the specified time period.
Still on Hold Amendment Trials Number of trial amendments CTRO placed on hold in the specified time period, that are still on hold at the time of report generation.
Abstracted Amendment Trials Number of trial amendments CTRO abstracted in the specified time period.
TSR Sent Amendments Number of trial amendments for which TSRs have been sent in the specified time period.
Accrual Submitted Trials Number of trials for which accrual has been submitted in the specified time period.
Accrual Validation Passed Count Number of trials that passed the validation checks when the batch file was received and processed in CTRP for the specified time period.
Trials Updated All Trials Number of trial updates submitted in the specified time period.
Trials Update Complete Trials Number of Complete trials for which updates have been submitted in the specified time period.
Trials Update Abbreviated Trials Number of Abbreviated Trials for which updates have been submitted in the specified time period.
