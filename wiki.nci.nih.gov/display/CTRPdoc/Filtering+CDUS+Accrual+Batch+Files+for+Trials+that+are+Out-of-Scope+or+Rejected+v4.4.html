CTEP provides a set of CDUS files for accrual batch upload into CTRP quarterly. A subset of these files contain trial records that trigger errors upon upload to the CTRP because they were closed prior to January, 2009, have been rejected during trial processing previously, or are missing in the CTRP. Files corresponding to these trials are flagged with errors at each subsequent quarterly submission.To minimize errors, Super Abstractors can designate each trial as either Out of Scope, or Rejected.During each subsequent quarterly submission, the system compares the batch files with trials in the list, and processes them as follows:
  • If it finds a match with a trial that has been designated either Out of Scope, or Rejected, the system disregards the trial's accrual batch file and removes the trial from the list.
  • If it finds a match with a trial that has no designation (i.e., left blank), the system submits the trial's accrual batch file. After successful submission, the system removes the trial from the list.
How to Filter CDUS Accrual Batch Files
  1. On the menu, click Accruals Out-of-Scope Trials.The Accruals Out-of-Scope page lists trials associated with CDUS accrual batch files that are missing or rejected.By default, trials that have a processing status Rejected, are designated as such in the list of flagged files.
  2. To remove trials from the list permanently, in the CTRO Action column, select either Rejected or Out-of-Scope.
  3. To process the CDUS trial accruals, leave the field in the CTRO Action column blank.
  4. Click Save.
