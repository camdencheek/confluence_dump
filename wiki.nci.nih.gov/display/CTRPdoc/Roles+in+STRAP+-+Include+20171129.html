The following table describes the roles available in STRAP:
Role Description
Cancer Center Users with this role can view only one Automated Report, the CTRP Data Table 4 report for their own Cancer Center. For instructions, refer to Running CTRP Data Table 4 Reports .
Administrator Users with this role can view the following features:
  • A Home page.
  • Dashboards. For instructions, refer to Configuring the Admin Dashboard and Types of Trials Included in the STRAP Dashboard .
  • Automated Reports. For instructions on running the CTRP Data Table 4 report for any Cancer Center, refer to Running CTRP Data Table 4 Reports . For instructions on running any internal report, refer to NCI CTRP Internal Reports Reference Guides .
  • Search Tool.
