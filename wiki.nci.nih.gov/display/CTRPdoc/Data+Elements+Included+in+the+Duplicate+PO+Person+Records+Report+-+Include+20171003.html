The following table describes the data elements displayed in the Duplicate PO Person Records report for each person record:
Data Element Description
PO ID Unique ID for the person, assigned by CTRP.
First Name The person's first name.
Last Name The person's last name.
CTEP ID Unique ID for the person, assigned by the Cancer Therapy Evaluation Program (CTEP).
Email The person's email address.
Phone The person's phone number.
