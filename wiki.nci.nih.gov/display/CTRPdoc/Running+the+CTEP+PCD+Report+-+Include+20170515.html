This page tree describes how to run the CTEP PCD report based on information available in the Clinical Trials Reporting Program (CTRP). This report is intended for Clinical Trials Reporting Office (CTRO) and Coordinating Center for Clinical Trials (CCCT) users. This report identifies CTEP Trials where the Primary Completion Date (PCD) has expired or is about to expire (within the next 90 calendar days).
