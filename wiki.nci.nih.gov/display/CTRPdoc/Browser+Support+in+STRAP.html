We test STRAP only on Firefox. For the following browsers, STRAP should work without issue:
  • Firefox
  • Chrome
  • Safari
However, please note that it has been officially tested only on Chrome. Also, STRAP does not support Internet Explorer.If you discover a browser-related issue, please contact us at ctrp_support@nih.gov . Please include the following information:
  • Your browser name and version
  • Your operating system version
  • When possible, a screen shot illustrating the difficulty
