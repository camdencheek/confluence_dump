The system imports Industrial and Consortia trials registered in ClinicalTrials.gov into the CTRP database nightly. Because these trials have been assigned ClinicalTrials.gov Identifiers (NCT IDs), the system attempts to match them with trials currently registered in the CTRP that also have NCT IDs:
  • If the system cannot find a trial in the CTRP with the same ClinicalTrials.gov Identifier, the system imports the trial as a new trial.
  • If the system finds a matching ClinicalTrials.gov Identifier, the system updates the CTRP record with the data imported from ClinicalTrials.gov. (If the system finds a match between a trial in ClinicalTrials.gov and a rejected one in the CTRP, the system will not import the trial. For instructions, refer to Deleting NCT-IDs from Rejected Trials .)
  • For additional information regarding the ClinicalTrail.gov Import Update click here .
 The system does not import personal information and does not include it in the update.You can change data that the system has imported from ClinicalTrials.gov.
