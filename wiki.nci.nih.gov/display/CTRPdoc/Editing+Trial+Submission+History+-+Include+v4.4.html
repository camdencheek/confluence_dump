To edit amendment submission details:
  1. Search for the trial of interest. For instructions, refer to Searching for Trials in PA .
  2. In the search results, click the NCI Trial Identifier link for that trial. The Trial Identification page appears.
  3. On the Trial Identification page, check out the trial. For instructions, refer to Checking In and Checking Out Trials . (This checkout step is optional for Super Abstractors.)
  4. On the Trial Overview menu, click Trial History. The Trial History Information page appears.
  5. In the Action column, click the Edit (pencil) icon. The Edit Trial History page appears. Edit Trial History Page
  6. In the Amendment Reason Code drop-down list, select a valid reason code. The following table lists valid reason codes.
    Reason Code Definition
    Administrative Amendment submitted for errors or changes to non-scientific data Example: A lead organizationâ€™s address changes
    Scientific Amendment submitted for errors or changes to the trial design Example: A trial advances from Phase 1 to Phase 2
    Both Amendment submitted for both administrative and scientific reasons
  7. If necessary, update the value in the Amendment Number field. If the trial submitter specified a value as the amendment number while amending the trial, this field displays that value. (This value may be a text string, whereas the submission number is numeric.)
  8. In the Amendment Date field type or select the amendment date.The amendment date is generally entered by the trial submitter.
  9. Click Save.
