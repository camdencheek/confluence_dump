Submitted trials that are ready for validation are listed in search results tables along with trials in other stages of the processing life cycle (Processing Status). You can list only those trials that require validation by limiting your search to trials that have the processing status, Submitted.  Or, you can sort an existing list of trials by the Action column.How to Search for Trials that Require Validation
  1. Follow the instructions in Searching for Trials , using only the Processing Status criterion, Submitted.
  2. Select one of the following trial categories from the Search by Trial Category drop-down list on the Trial Search page:
    • To limit your search to Complete trials, select Complete. -or-
    • To limit your trial search to Abbreviated trials, select Abbreviated. Trials that require validation are listed in the search results table.
  3. In the Action column for the trial you want to validate, click Validate. Or, click the NCI Trial Identifier link. The Trial Identification page appears with information about the selected trial. The Validation menu appears with different options depending on whether the trial is Complete or Abbreviated.  Validation Menu for Complete Trials Validation Menu for Abbreviated Trials
How to Sort Search Results by Validation Status
  1. In a search results table from any search, sort the trials by Action. To do so, click the Action column heading. The trials are sorted by Action, in ascending order. Search Results – Sorted by Action
  2. Click the Action column heading again to reverse the sort order to display validation-ready trials at the top of the table.
