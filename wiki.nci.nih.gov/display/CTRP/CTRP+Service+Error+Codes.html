The following is a list of error codes returned by the CTRP Services.
Code Description
PA_GENERAL Unknown or unspecified error code
PA_SYS_001 A unrecoverable system error has occured.
PA_USER_001 The user is not an authorized CTRP user
PA_USER_002 The user is not an owner of the study protocol
PA_DATA_001 The NCI trial ID provided is invalid
PA_DATA_002 The study protocol has not been abstracted and verified (i.e., it is not in the "Abstraction Verified Response"  or "Abstraction Verified No Response" status), so the XML file cannot be generated.
PA_XML_001 The generation of the XML file failed
