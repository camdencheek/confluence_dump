What you have learned?

  • List anything you learned about in this sprint (add your name after your comment)
  • I am getting the opportunity to learn more about URIs (Unique Resource Identifiers) that are a part of the Data Repository Service (DRS) which will ultimately help us to implement an "Export Files" button to the Cancer Genomics Cloud (CGC) ( Kuffel, Gina (NIH/NCI) [C] )
  • Writing frameworks needs a completely different mindset from writing application. Best practice for writing application might not be best for a framework. ( Ying, Ming (NIH/NCI) [C] )
  • MVC pattern for configuration. ( Rana, Ambar (NIH/NCI) [C] )
  • I learned a lot about dependency upgrading tools and processes as well as best practices associated with this that can be implemented into our future work processes ( Mueller, Austin (NIH/NCI) [C] )
  • I learned about access-token based authentication and how it can benefit Bento when we implement token-based authentication in the next Sprint ( Mueller, Austin (NIH/NCI) [C] )
