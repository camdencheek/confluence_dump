The C-BIBOP project is focused on cloud-based challenges and workflows. C-BIBOP provides infrastructure to run challenges in which participants upload their algorithms as Docker containers. It also supports reprducible science through worksheets that share pipelines for algorithms and processes.  An example of a pipeline might include algorithms for lung-field segmentation, lung-nodule segmentation feature extraction and classification. The following videos demonstrate the use of the CBIBOP platform for managing challenges:
