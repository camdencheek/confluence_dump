The Generating Predictions page describes how to generate predictions from a model asset collection (and optionally evaluate the model).
  • When generating predictions from an MT-CNN model, you can select one of the following file types as an input file:
    • A GDC manifest file (TXT).
    • A pathology report (TXT or PDF).
  • When evaluating the model, you can select a CSV outcome file. Make sure the number of rows of data in the outcome file matches the number of rows of data in the input file.
