If you generated the prediction files within an asset, you can manage which groups can access those prediction files.
  1. Log in, as described in Logging In .
  2. Perform a search, as described in Finding Data .
  3. In the search results, click the link for the asset of interest. The Asset Details page appears. If you have generated predictions in this asset, or if you belong to an access group associated with the predictions in this asset, a Predictions tab lists those prediction files.
  4. Click the Predictions tab.
  5. In the row you want to modify, click the edit icon. The Edit Prediction Access dialog box appears.
  6. In the Prediction Access Groups field:
    • If you do not want the public to have access to the prediction files, clear the Public checkbox and select one or more groups from the list.
    • To make the prediction files available to all logged-in user accounts with the Upload privilege, select Public.
  7. Click Update. A message confirms the update.
