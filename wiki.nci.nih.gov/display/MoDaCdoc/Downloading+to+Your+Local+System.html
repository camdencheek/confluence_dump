If you have a MoDaC user account, you can download one file at a time to your local system.
  1. Log in, as described in Logging In .
  2. Search for the data of interest, as described in Finding Data .
  3. In the search results, click the link for the asset of interest. The Asset Details page appears.
  4. If the asset has subfolders, navigate to the file of interest. (Click each folder to open it.)
  5. In the row for the file you want to download, click the download file icon. The Download page appears.
  6. Select Local System.
  7. In the Download File Name field, you can keep the existing file name or specify a new file name.
  8. Click Download. Follow the prompts of your operating system to complete the download. The system transfers the file based on your selections.
