If the following criteria are met, you can delete a file or an empty collection:
  • You have permission to delete it in one of the following ways:
    • Via ownership: You created it.
    • Via group permissions (as described in Managing Edit Permissions ) and delete permission. (To request permission to delete data, select About > Contact. For details, refer to Contacting Us .)
  • The file or collection is not public. For information, refer to Sharing Collections with Groups .
  • The file or collection is less than 90 days old.
If you want to delete data that does not meet the above criteria, select About > Contact. For details, refer to Contacting Us .Contents
