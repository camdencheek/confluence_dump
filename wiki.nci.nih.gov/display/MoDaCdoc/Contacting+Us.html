1. Open a supported browser:
    • Chrome
    • Firefox
  2. Navigate to the following portal: http://modac.cancer.gov/  The home page appears.
  3. Select About > Contact (or in the footer, select Contact Us). The Contact Us page appears.
    • If you have logged in, as described in Logging In , the system automatically specifies your name and email address.
    • Otherwise, specify your name and email address.
  4. Type your message. To copy one of the following examples, click the icon next to it:
    • I would like permission to upload data.
    • I would like to change the email address in my MoDaC user account.
    • I would like to change the groups listed on the Manage Account page for my MoDaC user account.
    • I have a question about preparing a metadata file to perform a bulk upload. (The Uploading Multiple Assets from a Globus Endpoint page might help.)
    • My upload is failing and I don't know why. I tried clicking retry, but the issue persists.
    • My download is failing and I don't know why. I tried clicking retry, but the issue persists.
    • I would like to change which groups can access a collection. (The Managing Group Access to an Existing Collection page might help.)
    • I would like to delete data. (The Deleting Data page family might help.)
  5. Respond to the reCAPTCHA prompt.
  6. Click Submit. The system sends your message to the MoDaC team.
