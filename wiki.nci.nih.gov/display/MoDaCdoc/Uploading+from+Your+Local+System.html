If the following criteria are all true, you can upload one file at a time from your local system to an asset in MoDaC (or to a subcollection of that asset):
  • You have a MoDaC user account with upload permission.
  • You are member of a group that has permission to edit that asset (or you created that asset).
  • The file is 2 GB or less.
When you create a collection or upload data, you become the owner of that collection or data. (Ownership cannot be changed.)
  1. Log in, as described in Logging In .
  2. Click Upload in the header. The upload page appears.
  3. Navigate to the asset:
    1. Select an existing program or register a new one.
    2. Select an existing study or register a new one.
    3. Select an existing asset or register a new one.
    For instructions, refer to Adding a Collection .
  4. You have the following choices:
    • To upload to the asset, click Upload Files to Asset.
    • To upload to a subcollection, navigate to that subcollection and click the upload icon next to it.
    The Upload Files page appears.
  5. Click Select a File.
  6. Specify the data and metadata:
    • Click Choose File. Navigate to and select your file. The filename appears next to the Choose File button.
    • If you want to add metadata, click Add Metadata. The system adds a row to the metadata table. Specify an attribute name and an attribute value. (Repeat for additional attributes. To delete an attribute, click the X.) Keep in mind the following points:
      • The character limit for each metadata value is 2700.
      • If an attribute value begins with http or https, MoDaC displays it as a hyperlink in read-only fields.
  7. Click Upload. The system transfers the file based on your selections. A message confirms the upload.
