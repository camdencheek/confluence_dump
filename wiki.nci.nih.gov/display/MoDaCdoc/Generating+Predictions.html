If you are a researcher, you can use your input and outcome files to find models in MoDaC that suit your purposes. If all of the following conditions are true, you can generate predictions from a model asset collection:
  • You have a MoDaC user account with upload permission.
  • You belong to a group that can modify that model.
  • That model is available for analysis.
To generate predictions:
  1. Log in, as described in Logging In .
  2. Perform a search for data, as described in Searching for Data You Can Edit . In the Filters panel, under Is Model Deployed, select Yes. The search results appear.
  3. In the search results, click the link for the asset of interest. The system displays the details page for that asset. For information, refer to Exploring Details of an Asset .
  4. In the Asset Files section of the page, click Generate Predictions. The Generate Predictions dialog box appears. The upload options available in this dialog box depend on various factors:
    • Upload GDC Data: This option is available for all models.
    • Upload Input File: This option is available only for models that support external data.
    • Select from MoDaC Reference Datasets: This option is available only when the current model is an applicable model in the reference dataset, as described in Adding a Reference Dataset .
  5. If necessary, select one of the above options.
    1. If you chose Upload GDC Data or Upload Input File:
      1. Click the first Choose File button. Navigate to and select an input file to generate predictions. For details, refer to one of the following pages:
        • About Files for an MT-CNN Model
        • About Files for a TC1 Model
        The filename appears next to the Choose File button.
      2. Optionally click the second Choose File button. Navigate to and select an outcome file to evaluate the model. For details, refer to one of the above pages. The filename appears next to the Choose File button.
    2. If you chose Select from MoDaC Reference Datasets:
      1. If the reference dataset you want to select is not in the list, add the model to that dataset as an applicable model. For details, refer to Adding a Reference Dataset .
      2. In the list of reference datasets, select one or more datasets.
  6. Click Submit. The system processes your request.
  7. When the system displays the task ID, consider clicking that link to view the status of the request. For instructions, refer to Viewing Status .
When the system completes the prediction task, you can view the files in the Asset Details page, in a Predictions tab. For details on actions you can take with these files, refer to the following pages:
  • Exploring Predictions in an Asset
  • Sharing Predictions with Groups
  • Downloading Data
  • Deleting Files
