If you have a MoDaC user account with upload permission, you can manage which groups can access your collections.
  1. Start adding a collection as described in Adding a Collection .
  2. When specifying attribute values for the new collection, decide whether to change the access_group attribute: By default, the collection is public. If you do not want the public to have access to the data, clear the PublicÂ checkbox and select one or more groups from the list.
  3. When you have finished specifying information for the new collection, click the Register button (such as Register Program).
