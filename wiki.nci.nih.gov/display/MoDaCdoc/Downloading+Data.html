If you have a MoDaC user account, you can download data:
  • You can download one file at a time to your local system.
  • If you have write access to a Globus endpoint, AWS S3 bucket, Google Drive folder, or Google Cloud bucket, you can download one or more files to it.
  • If you have a MoDaC user account with upload permission, you can download multiple assets.
  • If an asset has subfolders, you can download them, one at a time.
  • If you have requested predictions, when they are ready for download, you can download them from the Status page or Asset Details page. (For details, refer to Viewing Status Â and the following pages.)
You can also download the metadata for one or more files in an asset to your local system.Contents
