If all of the following conditions are true, you can perform model analysis on a reference dataset:
  • You have a MoDaC user account with upload permission.
  • You belong to a group that can modify that dataset.
To generate predictions:
  1. Log in, as described in Logging In .
  2. Upload an input file and an outcome file to the reference dataset, as described in Uploading Data .
  3. Perform a search for data, as described in Searching for Data You Can Edit . The search results appear.
  4. In the search results, click the link for the reference dataset. The system displays the details page for that asset. For information, refer to Exploring Details of an Asset .
  5. In the Asset Files section of the page, click Perform Model Analysis. The Perform Model Analysis dialog box appears.
  6. Select the model (or models) you want to use for analysis. Click Submit. The system processes your request.
  7. When the system displays the task ID, consider clicking that link to view the status of the request. For instructions, refer to Viewing Status .
When the system completes the prediction task, you can view the files in the Asset Details page, in a Predictions tab. For details on actions you can take with these files, refer to the following pages:
  • Exploring Predictions in an Asset
  • Sharing Predictions with Groups
  • Downloading Data
  • Deleting Files
