If you have a MoDaC user account with upload permission, you can manage which groups can edit the metadata for your collections (at the asset level or higher). The groups you specify can also add or delete files or subcollections.
  1. Start adding a collection as described in Adding a Collection .
  2. In the Permissions field,Â select one or more groups from the list. (If the Permissions field is empty, only you, as the person who created the collection, can edit the metadata, add files, or add subcollections.)
  3. When you have finished specifying information for the new collection, click the Register button (such as Register Program).
