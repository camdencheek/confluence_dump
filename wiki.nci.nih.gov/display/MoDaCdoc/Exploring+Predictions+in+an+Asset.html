1. Log in, as described in Logging In .
  2. Search for the data of interest, as described in Finding Data .
  3. In the search results, click the link for the asset of interest. The Asset Details page appears. If you have generated predictions in this asset, or if you belong to an access group associated with the predictions in this asset, a Predictions tab lists those prediction files.
The following table describes the actions you can take in the list of prediction files.
To do this ... Do this ...
Select the number of rows per page In the Rows per Page box, select a number. The minimum number of rows per page is 10 and the maximum is 100.
Move to the previous page of results Click the previous icon.
Move to the next page of results Click the next icon.
View metadata for a file Next to the file of interest, click the Metadata icon ( ).
Copy the path for a file Next to the file of interest, click the Copy Path icon ( ). The system copies the path to your clipboard.
Download a row In the row of interest, click the Download Model Analysis Files icon ( ). The Download page appears. For details, refer to Downloading Data .
Download multiple rows Select the Batch Select checkbox. (To exclude a row from download, click the checkbox for that row.) Click Download Selected Files. The Download page appears. For details, refer to Downloading Data .
Edit group access to a row (if available) In the row of interest, click the Edit Prediction Access icon ( ). For details, refer to Sharing Predictions with Groups .If you generated the prediction files, you can edit the group access to those files.
