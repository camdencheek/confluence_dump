If you have a MoDaC user account, you can view the following statuses:
  • You can view the status of your Globus, AWS S3, Google Drive, and Google Cloud downloads.
  • If you have upload permission, you can view the status of your Globus, AWS S3, Google Drive, and Google Cloud uploads.
  • If you have generated predictions from a model or a reference dataset, you can view the status of that task.
To view statuses:
  1. Log in, as described in Logging In .
  2. Click Status in the header. A status page appears.
    • The Transfer Tasks tab lists download and upload tasks. For downloads, the information in the Asset Identifier column depends on how you downloaded the data.
      • If you downloaded from the search results list, the asset identifier appears.
      • If you downloaded a file from the Asset Details page, the individual file name appears.
    • The Prediction Tasks tab lists prediction tasks.
You can perform the following the actions in the list of tasks:
To do this ... Do this ...
Select the number of tasks per page In the Rows per Page box, select a number. The minimum number of tasks per page is 10 and the maximum is 100.
Move to the previous page of tasks Click the previous icon.
Move to the next page of tasks Click the next icon.
Sort a column Click the column header.
View details about an asset In the Asset Identifier column, click the link. The Asset Details page appears. For details, refer to Exploring Details of an Asset . (New files appear in the Asset Details page one hour after they were created.)
In the Transfer Tasks tab: View program, study, and asset identifier for a transfer Click the information icon in the Asset Identifier column.
In the Transfer Tasks tab: View the error message for a failed transfer. Click the information icon in the Transfer Status column.
In the Transfer Tasks tab: Retry a failed Globus transfer In the Transfer Status column, click the retry icon. For details, refer to Retrying a Failed Upload or Download .
In the Prediction Tasks tab (if available): Download predictions In the Status column, click the download icon. For details, refer to Downloading Data .
