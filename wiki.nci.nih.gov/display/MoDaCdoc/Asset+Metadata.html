Metadata Name Metadata Description Metadata Type
Asset Name Name the asset. Mandatory, free text
Asset Description Describe the asset. Spell out each abbreviation at first use. Mandatory, free text
Asset Identifier Specify a unique, permanent identifier for the asset. Use only alphanumeric characters, underscores (_), or hyphens (-). Mandatory, not modifiable once created
Asset Type Valid values for Asset Type are dataset or model. Mandatory, specified before asset creation
DOI Specify the Document Object Identifier of the publication associated with the dataset, if available. Optional, free text
POC Name Name the point-of-contact person for the asset in the following format: Lastname, Firstname Optional, free text
POC Email Specify the point-of-contact email address for the asset. Optional, email address format
