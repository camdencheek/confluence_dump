If you are a researcher, you can use your input files to generate predictions and your outcome files to find models in MoDaC that suit your purposes. The following pages describe the input and outcome files for each model:
  • About Files for an MT-CNN Model
  • About Files for a TC1 Model
When your files are ready, choose one of the following methods:
  • In the Asset Details page for a model, you can upload files and generate predictions. In this case, the system requires an input file and the outcome file is optional. For instructions, refer to Generating Predictions .
  • You can create a reference dataset in MoDaC, upload files to it, and then perform a model analysis. In this case, the system requires an input file and an outcome file in the reference dataset. For instructions, refer to the following pages:
    1. Adding a Reference Dataset
    2. Performing Model Analysis Using a Reference Dataset
