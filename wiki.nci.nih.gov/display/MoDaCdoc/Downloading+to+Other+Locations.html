To download data from MoDaC to a Globus endpoint, AWS S3 bucket, Google Drive folder, or Google Cloud bucket:
  1. Log in to MoDaC, as described in Logging In .
  2. Search for the data of interest, as described in Finding Data .
  3. In the search results, specify data for download:
    • To download part of an asset, click the link for the asset of interest. The Asset Details page appears. (If the asset has subfolders, navigate to the file of interest: Click each folder to open it. If the asset has prediction files that you want to download, click the Predictions tab.) Select one or more files for download:
      • To download one file, click the download file icon in that row.
      • To download one folder, click the download folder icon in that row.
      • To select all files at the asset level, select the Batch Select checkbox. (To deselect one file, click the checkbox in that row.) Click Download Selected Files.
    • To download an entire asset, select the radio button or checkbox for the asset of interest. (This option is not available for large assets: 1 TB or larger.) Click Download Selected Assets.
    The Download page appears, listing the selected file or files. (To deselect one file, click the checkbox in that row.)
  4. Select the option you want to use and refer to the following pages for additional instructions:
    • Downloading to a Globus Endpoint
    • Downloading to an AWS S3 Bucket
    • Downloading to Google Drive
    • Downloading to Google Cloud
