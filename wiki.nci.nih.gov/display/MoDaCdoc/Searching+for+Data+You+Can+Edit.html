1. Open a supported browser:
    • Chrome
    • Firefox
  2. Navigate to the following portal: http://modac.cancer.gov/  The home page appears.
  3. Log in, as described in Logging In .
  4. Click Search in the header. The Search page appears with the first page of all results.
  5. In the Search Results panel, click Assets I Can Edit. The system lists only the collections you can edit. For details, refer to Editing Metadata .
  6. (Optional) Continue narrowing results as described in Searching for Data .
The system lists all matching results. For details, refer to Using Search Results .
