Protégé Plug-ins

As the [Protégé] framework has developed over the years, its pluggable nature has garnered plug-ins that add additional value to the framework from various organizations. Here is a list of some you might find useful.
  • NCI_Protégé - NCI Protégé is a collection of Protégé plug-ins developed for the NCI-specific editing environment.
  • FCAView Protégé Tab Plug-in - FCAView Tab plug-in provides Protégé users a novel way to visualize their ontologies using the formal concept analysis (FCA) technique.
