Protégé 4 (build 113) Release

June 16, 2009 The highlights of this release are:
  • Protégé-OWL editor built on top of the open source OWL API
  • Support for OWL 2
  • Direct, in-memory connection to Pellet and FaCT++ reasoners
  • Support for editing multiple ontologies
  • Creation, import, and export of user configurable tabs
For further information, visit: Protégé 4 Release Announcement Protégé 4 Feature Overview Protégé 4 Documentation Protégé 4 Download Protégé 4 Getting Started

Protégé 3.4.2 (build 562) Release

December 17, 2009 Highlights of this update include:
  • Updates to Proégé Client-Server
    • Modifications to the metaproject
    • Modifications to the Server Admin Panel
    • Modifications to the API
  • Updates to Core Protégé
  • Updates to Protégé-OWL
  • Updates to Collaborative Protégé & Change Management
    • Modifications to the Changes & Annotation Ontology (ChAO)
    • Collaborative Protégé now supports the archival of notes and discussion threads
    • The default notes view will show an "archived" check box and a drop-down list for selecting the status of a note
    • The number of notes attached to a class is shown in the class tree next to the notes icon
    • A status filter for notes and changes has been added to the filter combo box
    • Made performance improvements by using a cache of note counts and ontology components
  • New Bundled Plug-ins
    • Axiome
    • BioPortal Reference Plugin
    • Lucene Query
  • Updates to Existing Bundled Plug-ins
For further information, visit: Protégé 3.4.2 Release Announcement Protégé 3.4.2 Download

Protégé 3.4.7 Release

July 15, 2011 The highlights of this release are:
  • Added additional SQWRL operators to support more elaborate query result selection:
    • selecting a subset of results
    • retrieving all elements from a collection.
  • Updated SQWRL Query API.
  • Updated the BioPortal Reference plug-in to store the short term ID properly.
For further information, visit: Protégé 3.4.7 Release Announcement Protégé 3.4.7 Download
