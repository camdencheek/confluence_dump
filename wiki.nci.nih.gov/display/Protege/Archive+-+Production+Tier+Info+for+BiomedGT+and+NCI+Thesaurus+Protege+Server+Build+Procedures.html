Production Tier Info for BiomedGT and NCI Thesaurus Protégé Server Build Procedures

Production builds usually involve transferring a tested version of a build from a lower tier to production (usually the QA tier), using a secure shell client. Upon transferring the folders, careful configuration will have to be performed in order to have the new version run smoothly. Follow instructions for Protege Software Updates, or Protege Software Updates & Configuring a New Database Project if there is a change to the production database schema.The directory structure in Production differs from the other tiers. The directory structure also differs within production between the BiomedGT and NCI Thesaurus instances, but that's not all. The username to access either instance also differ.
