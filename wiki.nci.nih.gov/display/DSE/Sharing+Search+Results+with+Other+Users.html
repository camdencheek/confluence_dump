You can share with other users a series of investigations in the CSSI DCC Portal. For example, if you want to share all investigations that mention the keyword CTC, share the following URL: https://cssi-dcc.nci.nih.gov/cssiportal/search/?q=CTC  If you click the above link, the Search Investigations page appears with CTC in the Search box and with results of that search. Keep in mind the following points:
  • Make sure the q in the URL is lowercase.
  • If your search term is multiple words, replace each space with %20 as in the following example: https://cssi-dcc.nci.nih.gov/cssiportal/search/?q=Circulating%20Tumor%20Cell
  • If your user account has a default saved search, a search term in the URL (?q=<value> as in the above example) overrides the default search.
  • If you use an asterisk (*) as the search term, or if you omit the search term entirely, the Search Investigations page lists all results:
    • Example with an asterisk: https://cssi-dcc.nci.nih.gov/cssiportal/search/?q=*
    • Example that omits the search term: https://cssi-dcc.nci.nih.gov/cssiportal/search/?q=
  • If you use the Search box as described in Performing a Basic Search , this search overrides any search terms in the browser address bar.
  • If you visit another page, the system displays the URL of the new page, effectively clearing the search from the URL.
