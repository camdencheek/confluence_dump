1. Create or import an investigation, as described in one of the following pages:
    • Creating an Investigation
    • Importing an Existing Investigation
  2. In the Node panel, select the node of the study or assay in which you intend to edit one or more process steps.
  3. In the Visualization panel, you have the following choices:
    • If you want to edit one process step, select the Process view and then select the process step icon. The system highlights the selected icon in green. A new panel appears. In that panel, edit the values for the selected process step.
    • If you want to edit multiple process steps, select the Table view. In the columns that correspond with the process steps of interest, edit the values.
  4. Consider saving your work, as described in one of the following pages:
    • Saving an Investigation
    • Saving a Node as a Template
