When you create a folder (as described in Uploading ISA Archives ), you are automatically the owner of that folder. You can add additional owners as needed. All owners must be registered CSSI-DCC users with the uploader role. It is a best practice to have only one folder owner; if you share ownership, the other owner could accidentally delete you.Each owner of a folder can perform the following tasks:
  • Upload an additional investigation version to that folder
  • Rename an investigation folder
  • For each study or assay in an investigation version, either give all users access or restrict access to one or more groups
  • Preview an investigation version
  • Request open access for an investigation version
  • View or edit the metadata of an investigation version
  • Add an owner to the folder or remove an owner from it
