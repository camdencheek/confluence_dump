1. Open the edit metadata page for an investigation, as described in Editing Investigation Metadata .
  2. If you are adding or editing a study contact, in the Studies panel, select the study you want to edit. Metadata fields for that study appear.
  3. You can add a publication, edit an existing publication, or both:
    • To add a publication, click New Publication. Specify publication information and click Add.
    • To edit a publication, click the edit icon in the row for that publication. Modify the publication information as needed. Click Update.
    In each case, you can specify the following information:
    • PubMed ID (If youÂ specify a valid PubMed ID, the system automatically populates other fields for this publication.)
    • Title
    • Author List
    • Status
    • DOI
    • Status Term Accession Number
    • Status Term Source Reference
  4. Click Save. The system saves your changes.
