While previewing an investigation, you can make the investigation version available to all CSSI DCC users by requesting open access for this version. If the investigation contains an access-controlled file, you can also review the access controls, by following a link.
  1. Log in to the CSSI DCC Portal, as described in Logging In to the CSSI DCC Portal .
  2. Select Investigations > Upload. The Upload ISA Archives page appears.
  3. For the folder of interest, click Open. The list of investigation versions for that folder appears.
  4. If you have uploaded multiple versions of the file to the folder, decide which version of the investigation you want to preview.
  5. Click the Preview button next to that version. The Open Access Preview page appears. The following screenshot shows the wording on this page if the investigation is access controlled.
  6. Review the uploaded file by exploring the investigation details, as described in Exploring Investigation Details .
At this point, you can request open access for the investigation version. For instructions, refer to Requesting Open Access .
