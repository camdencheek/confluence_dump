The Metadata Utility can facilitate the creation of the tab-delimited text files for an ISA archive. This page describes how to import these files for continued development. The terminology in this utility reflects that these files represent an investigation.
  1. Start the Metadata Utility, as described in Starting the Metadata Utility .
  2. In the Action panel, click Import ISA TAB Investigation. The Open dialog box appears.
  3. Navigate to and select your investigation file. The system requires a tab-delimited text file (extension txt) and the filename must begin with "i_" (such as i_investigation.txt). The Metadata Utility reappears. The following items appear in the Metadata Utility:
    • The Action panel displays options to validate or save the investigation. For instructions, refer to the following pages:
      • Validating an Investigation
      • Saving an Investigation
    • The Node panel displays various nodes, including a node for the investigation. For instructions on modifying these nodes, refer to Modifying an Investigation .
    • The Validation panel displays any validation errors. For instructions on processing such errors, refer to Validating an Investigation .
