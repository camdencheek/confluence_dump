You can edit the metadata of an investigation that you own.
  1. Log in to the CSSI DCC Portal, as described in Logging In to the CSSI DCC Portal .
  2. Select Investigations > Upload. The Upload ISA Archives page appears.
  3. For the folder of interest, click Open. The list of investigation versions for that folder appears.
  4. If you have uploaded multiple versions of the file to the folder, decide which version of the investigation you want to edit.
  5. Click the View/Edit Metadata button next to that version. The edit metadata page for that investigation version appears. The following table provides instructions for various actions you can take:
    Action Instructions
    Edit the title Click the edit icon next to (or below) the title. Modify the title as needed. Click Save.
    Edit the description Click the edit icon below the description. Modify the description as needed. Click Save.
    Add or edit a contact Refer to Adding or Editing a Contact .
    Delete a contact If the investigation has more than one contact, you can delete one by clicking the trash icon in that row. Click Save.
    Add or edit a publication Refer to Adding or Editing a Publication .
    Delete a publication If the investigation has more than one publication, you can delete one by clicking the trash icon in that row. Click Save.
    Add or edit an ontology source reference Refer to Adding or Editing an Ontology Source Reference .
    Delete an ontology source reference If the investigation has more than one ontology source, you can delete one by clicking the trash icon in that row. Click Save.
  6. In the Studies panel, select the study you want to edit. Metadata fields for that study appear. The following table provides instructions for various actions you can take:
    Action Instructions
    Edit the title Click the edit icon next to (or below) the title. Modify the title as needed. Click Save.
    Edit the description Click the edit icon below the description. Modify the description as needed. Click Save.
    Add or edit a contact Refer to Adding or Editing a Contact .
    Delete a contact If the study has more than one contact, you can delete one by clicking the trash icon in that row. Click Save.
    Add or edit a publication Refer to Adding or Editing a Publication .
    Delete a publication If the study has more than one publication, you can delete one by clicking the trash icon in that row. Click Save.
Each time you click Save, the system saves your changes.To return to the Upload ISA Archives page, click Return at the top of the edit metadata page.
