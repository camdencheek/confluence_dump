1. Create or import an investigation, as described in one of the following pages:
    • Creating an Investigation
    • Importing an Existing Investigation
  2. In the Node panel, select the node (investigation, study, or assay node) in which you intend to add an element or element group.
  3. In the Element panel, navigate to the location where you intend to add an element or element group. (To expand or collapse an element group, click the header.) Hover below an existing element or element group. Green "Add item ..." text appears.
  4. Click the Add item link. What happens next depends on the location:
    • Within a group that contains entry groups, a new entry appears in the specified location.
    • Otherwise, the Add Element dialog box appears.
  5. If the Add Element dialog box appears, type a name and select an element type:
    • A group element is a header, under which you can later add more elements.
    • A text element is a field that allows a single line of text.
    • A long text element is a text area that allows multiple lines of text.
  6. In the Add Element dialog box, click Save. The new element or element group appears in the specified location.
  7. Consider saving your work, as described in one of the following pages:
    • Saving an Investigation
    • Saving a Node as a Template
