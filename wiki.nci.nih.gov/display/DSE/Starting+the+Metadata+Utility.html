1. Install the Metadata Utility, as described in Installing the Metadata Utility .
  2. In the downloaded and extracted folder, locate the "CSSI Metadata Utility.exe" file and run it. A splash screen appears while the utility is loading. Then the Metadata Utility appears. The Metadata Utility has the following panels:
    1. Action panel
    2. Node panel
    3. Element panel
    4. Validation panel
    5. Visualization panel
