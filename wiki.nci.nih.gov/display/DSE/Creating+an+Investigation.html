The Metadata Utility can facilitate the creation of the tab-delimited text files for an ISA archive. This page describes how to begin creating these files. The terminology in this utility reflects that these files represent an investigation.
  1. Start the Metadata Utility, as described in Starting the Metadata Utility .
  2. In the Action panel, click New Investigation. The New Investigation dialog box appears.
  3. Click New Empty Investigation. The Metadata Utility reappears. The following items appear in the Metadata Utility:
    • The Action panel displays options to validate or save the investigation. For instructions, refer to the following pages:
      • Validating an Investigation
      • Saving an Investigation
    • The Node panel displays a node for the investigation and an option to add a study. For instructions, refer to Modifying an Investigation .
    • The Validation panel displays validation errors. For instructions on processing these errors, refer to Validating an Investigation .
