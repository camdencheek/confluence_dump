You can save a study node or assay node as a template, and later create new studies or assays based on that template. This feature is helpful when you need similar content across multiple studies or assays. Study and assay templates are not transferable to another computer.
  1. If you are using a Mac, move the CSSI.app file to your Applications folder, as described in Installing the Metadata Utility .
  2. Create or import an investigation, as described in one of the following pages:
    • Creating an Investigation
    • Importing an Existing Investigation
  3. Add or edit items, as described in one of the following pages:
    • Adding Items
    • Editing Items
  4. In the Node panel, select the node of the study or assay that you want to save.
  5. In the Action panel, click Save Selected Node as Template. The Name Template dialog box appears.
  6. Type a name for the template. The name you choose will appear in the list of template options when you add a study or assay to an investigation. If you specify the same name as an existing template, your new template overwrites the previous one.
  7. Click Save. The Metadata Utility reappears.
For instructions on using the new template to add a node, refer to one of the following pages:
  • Adding a Study
  • Adding an Assay
