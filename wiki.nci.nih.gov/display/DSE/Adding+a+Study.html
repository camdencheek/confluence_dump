1. Create or import an investigation, as described in one of the following pages:
    • Creating an Investigation
    • Importing an Existing Investigation
  2. In the Node panel, click Add study. The Add study dialog box appears.
  3. From the Template Name list, select a template.
  4. Click Add study. The Metadata Utility reappears, with a new study node at the top of the list of studies in the Node panel. By default, the study file name is s_s0001.txt.
  5. Select the new study node. In the Element panel, scroll down to the Study File Name field and customize the filename. Consider giving your studies a sequence of filenames (such as s_s0001.txt, s_s0002.txt, and so on).
  6. In the Node panel, click another node. The file name change takes effect.
  7. Select the new study node again. In the Element panel, enter a value in each remaining required field, as indicated with an asterisk (*). Consider entering a value in each optional field.
  8. Consider saving your work, as described in one of the following pages:
    • Saving an Investigation
    • Saving a Node as a Template
