1. Open the edit metadata page for an investigation, as described in Editing Investigation Metadata .
  2. You can add an ontology source, edit an existing ontology source, or both:
    • To add an ontology source, click New Ontology. Specify ontology information and click Add.
    • To edit an ontology source, click the edit icon in the row for that ontology. Modify the ontology information as needed. Click Update.
    In each case, you can specify the following information:
    • Name
    • Description
    • File
    • Version
  3. Click Save. The system saves your changes.
