You can request to download data files from multiple investigations, studies, and assays. Your download options depend on whether you are logged in:
  • If you are not logged in: You must provide your email address so that you can receive an email message with a link to access and download the data.
  • If you are logged in: You can have the link sent to the address associated with your account.
To request multiple downloads
  1. Browse for or search for the investigation data that you want to download. For instructions, refer to Browsing Investigations  or Searching Investigation Data .
  2. Consider filtering the investigation data that you want to download. For instructions, refer to Visualizing and Filtering Data .
  3. Select the objects that you want to download. You can include in the download objects at any or all levels (one or more investigations, one or more studies, and/or one or more assays). You can include filtered or unfiltered content. Take one or both of the following actions:
    • In the browse results or search results, click the Select for Multi Download icon:
    • In the relevant details page, select the objects that you want to download. Click Add to Download. (If you no longer need the files you have selected for download on this details page, click Remove from Download.)
  4. When you have added all of the data files that you want to download, click the download icon in the menu bar. The Request Multiple Downloads dialog box appears, with a list of investigations. For each investigation, the dialog box indicates the number of files you have selected, the combined file size for those files, and whether you have filtered the content. The download options available depend on whether you are logged in to CSSI DCC.
  5. In the Request Multiple Downloads dialog box, consider the following options:
    • If you want to view the details page for a filtered investigation, click that investigation. (This feature applies only to filtered investigations.) To return to the Request Multiple Downloads dialog box, click the download icon in the menu bar again.
    • If you want to expand the list, click the triangle icon next to each item.
    • If you no longer need the files you have selected for download, you can clear some or all of your selections:
      • To remove all files: Click Clear Queue.
      • To remove only the files for an individual investigation, study, or assay: Click the relevant check box.
  6. When you are ready to complete the download, complete the following steps:
    1. Enter or confirm your email address.
    2. Click Request Download.
    3. If necessary, respond to the reCAPTCHA dialog box.
    The system responds in the following ways:
    • The Download Requested message appears.
    • DCC sends to the email address you specified a notification with "CSSI DCC Portal File Download Ready" as the subject line.
  7. When you receive a download notification message, download the content by clicking each link. For details, refer to Using an Email Notification or Download Summary .
