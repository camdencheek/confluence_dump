If you have a CSSI DCC user account with upload permission, you can download and install the Metadata Utility.
  1. Consider where to install the Metadata Utility. Study and assay templates are not transferable to another computer.
  2. Log in to the CSSI DCC Portal, as described in Logging In to the CSSI DCC Portal .
  3. Select Investigations > Upload. The Upload ISA Archives page appears.
  4. Scroll down to the tabs below the list of folders and select the Metadata Utility tab.
  5. Click the link for your operating system and download the file.
  6. Extract the executable file from the zip file.
  7. If you are using a Mac, move the CSSI.app file to your Applications folder. (This step allows you to use the functionality described in Saving a Node as a Template .)
For further instructions, refer to Starting the Metadata Utility .
