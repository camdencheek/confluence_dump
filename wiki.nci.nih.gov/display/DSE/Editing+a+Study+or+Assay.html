1. Create or import an investigation, as described in one of the following pages:
    • Creating an Investigation
    • Importing an Existing Investigation
  2. In the Node panel, select the node of the study or assay of interest.
  3. In the Element panel, enter a value in each required field, as indicated with an asterisk (*). Consider entering a value in each optional field.
  4. In the Visualization panel, select the view: Process or Table. The following table provides instructions for various options in each view:
    Action Process View Table View
    Add a process Refer to Adding a Process . Refer to Adding a Process .
    Add a process step Refer to Adding a Process Step . Refer to Adding a Process Step .
    Edit a process Refer to Editing a Process . Refer to Editing a Process .
    Edit a process step Refer to Editing One or More Process Steps . Refer to Editing One or More Process Steps .
    Delete a process step Click the X next to the process step icon. (Not available)
    Delete a process (Not available) Click the X next to the appropriate row.
  5. Consider saving your work, as described in one of the following pages:
    • Saving an Investigation
    • Saving a Node as a Template
