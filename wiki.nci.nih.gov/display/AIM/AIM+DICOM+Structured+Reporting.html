AIM DICOM Structured Reporting (DICOM SR) is a document that shows how AIM DICOM SR was created. Other DICOM engineers can use this information to implement their own DICOM library to read and write AIM DICOM SR.File in .rar compression format.rar is a proprietary archive file format that supports data compression, error recovery, and file spanning. Both proprietary and free programs are available for accessing such files. A tutorial on accessing .rar files, including a free extractor, is available .
