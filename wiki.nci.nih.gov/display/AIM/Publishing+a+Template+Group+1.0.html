You can publish a template group that you created in AIM Template Builder to the AIM Template Service so that others can find it or so that you can import it into another application.
  1. Click the Template Groups tab.
  2. Select a template group from the Groups panel.
  3. Below the left panel, click .
  4. Click Publish. A window opens, prompting you to log in to the AIM Template Service.
  5. Enter your password or log in as a different user and then click Login. You must have a caGrid account  to publish templates to the ATS. The steps that follow are unique to your configuration of the AIM Template Service.
