1. Click the Lexicons tab.
  2. Select the lexicon that you want to contain your new lexicon entry.
  3. Right-click the lexicon and select Create Lexicon Entry, or click in the toolbar. A panel for the new lexicon entry appears.
  4. Complete the fields to describe the lexicon entry.An explanation of the purpose of each field is shown in the table below.
    Field Description
    Code Meaning The name of the term you want to use as the lexicon entry
    Description A description of the term
    Code Value The value assigned by the coding scheme designator
  5. Click Save.
