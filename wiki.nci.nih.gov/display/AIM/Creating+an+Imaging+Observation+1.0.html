1. Click the Templates tab.
  2. In the Templates panel, select the template to which you want to add an imaging observation.
  3. In the Template Components panel, right-click the template name and select Create Imaging Observation. You can also find this option in the toolbar ( ). A new panel appears on the right and the toolbar below the Template Components panel changes to provide options for imaging observations, which are to add or delete an imaging observation ( ).
  4. In the panel on the right, select Annotator Confidence , if that attribute applies to the imaging observation.
  5. Click Save.
  6. Add one or more characteristics to the imaging observation. For example, to describe an image showing a cystic mass, the AIM template captures an imaging observation of mass and an imaging observation characteristic of cystic.
