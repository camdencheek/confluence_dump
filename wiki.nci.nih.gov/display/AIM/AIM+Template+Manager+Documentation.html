The AIM Template Manager is a web application that the AIM Template Builder replaced.Downloads are listed on the main page of this AIM wiki .AIM Template Manager is no longer availableThis user's guide explains how to use the AIM Template Manager (ATM), a web application that is no longer available. This application used an older version of the AIM template XML schema (version 1, revision 18). AIM Template Builder (ATB) replaces ATM. ATB is a Java application that you can install locally without a web connection. It uses the most recent release (version 1, revision 23) of the AIM template XML schema.Archived AIM Template Manager Documentation
