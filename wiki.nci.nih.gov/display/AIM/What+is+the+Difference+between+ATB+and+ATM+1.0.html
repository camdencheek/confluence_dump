What is the difference between the AIM Template Builder and the AIM Template Manager?This user's guide explains how to use the AIM Template Builder (ATB), a Java application that you can install locally without a web connection. It uses the most recent release (version 1, revision 23) of the AIM template XML schema. The AIM Template Manager is similar to ATB but is a web application that runs on the Internet. It uses an older version of the AIM template XML schema (version 1, revision 18) and will be available only until mid-2012.
