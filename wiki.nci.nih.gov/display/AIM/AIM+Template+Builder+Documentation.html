The AIM Template Builder (ATB) provides a tool to create concise and constrained templates consisting of a set of questions and possible answers when annotating images. The tool allows AIM templates to be created without requiring the template creator or extensive knowledge of the comprehensive AIM information model.Downloads are listed on the main page of this AIM wiki .AIM Template Builder Documentation
