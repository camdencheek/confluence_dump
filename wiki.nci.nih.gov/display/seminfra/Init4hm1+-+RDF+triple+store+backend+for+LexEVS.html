The goal of this requirement is leveraging the LexGrid model with Semantic Web standards and specifications. The LexRDF model will map the LexGrid model elements to corresponding constructs in W3C specifications. The data in the LexGrid will translate into RDF triples. LexEVS will query the RDF triples with Semantic Web query language.
