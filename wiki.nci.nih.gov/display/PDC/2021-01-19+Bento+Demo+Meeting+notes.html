More Details on Bento Anjan
  • The reference implementation of Bento can be found at: https://bento-tools.org/#/home . This is version 2.0.1
  • The Documentation on setting up Bento can be found at: https://cbiit.github.io/bento-docs/master/index.html Specifically, the Quick Start Tutorial gives the reader instructions on setting up Bento locally: https://cbiit.github.io/bento-docs/master/installation/bento-quick-start.html
  • The Data Model that drives this default Bento framework is described here: https://cbiit.github.io/bento-docs/master/data-and-modeling/Bento_TailoRx_Data_Dictionary.html
 We suggest that you use the TAILORx version of the model- as that is the most mature.
