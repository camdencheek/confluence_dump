1. Open a supported browser (Google Chrome) and navigate to the https://cadsr.cancer.gov portal. The home page appears.
  2. From the View menu, select Browse and then select Guest User Cart. The Guest User Cart page appears.
  3. In the Cart Name field, type the name of a cart you have saved previously.
  4. Click Apply Filter. The system displays the search results. By default, the system also hides the search criteria.
