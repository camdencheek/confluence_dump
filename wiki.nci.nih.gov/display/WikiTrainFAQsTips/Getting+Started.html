Anyone can view the NCI Wiki. In the Confluence header, click the icon to the left of the Search NCI Wiki box. Select Spaces. A list of the wiki spaces that you can view appears. To edit pages:
  1. If you do not have an account, refer to Getting an NCI Wiki Account .
  2. In the list of spaces, select the link for the space that interests you. The wiki home page opens for that space.
  3. Examine the links in the hierarchy of pages for this wiki space, in one of the following ways:
    • Some wiki spaces have a list of links reflecting a whole page family within the home page itself.
    • Some wiki spaces have such a list in the sidebar.
    • Some wiki spaces have a list of child pages at the bottom of every page.
    • All wiki spaces have this option: Click the Tools (...) icon above the page title, on the right side of the page. Select View in Hierarchy.
  4. Select the link for a page that interests you.
  5. At this point, you have the following options:
    • If you want to comment on the page, click Write a Comment at the bottom.
    • If you want to add a child page for a subtopic, click the Create Content icon in the upper right, select a template, and click Create.
    • If you want to edit the page, click Edit.
  6. Whether you are adding a comment, adding a page, or editing a page, type just as you would in Microsoft Word. For more information, refer to the following pages:
    • Adding a Page or a Blog Post
    • Editing a Page
  7. When you are finished, click Save.
  8. If you need more help, contact NCIAppSupport@nih.gov .
