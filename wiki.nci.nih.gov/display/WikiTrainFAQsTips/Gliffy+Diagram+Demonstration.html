To create a Gliffy diagram:
  1. In the page you want to change, click Edit.
  2. At the place on the page where you want the diagram to appear, type as much of the following as needed to open the macro browser. {gliffy
  3. Select the Gliffy Diagram macro. The Gliffy dialog box appears.
  4. Select an option and click Create. The Gliffy editing interface opens.
  5. Construct the diagram using the applicable options.
  6. Click Save.
  7. Click Back to Confluence. The diagram appears on the wiki page.
For more information, refer to the following pages:
  • Atlassian help pages:
    • Add-ons and integrations
    • Create Beautiful and Dynamic Pages
  • https://help.gliffy.com
