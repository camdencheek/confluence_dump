To find all content with a particular macro:
  1. Determine the macro name. You can do so by looking at a macro in Source Editor, or by looking it up in the available help for that macro. Here are some common examples:
    • htmlcomment
    • multiexcerpt-include
    • pagetree
  2. Perform a search as described in the following page: https://confluence.atlassian.com/conf613/search-964961041.html
  3. Enter a search string like the following, which finds all multiexcerpt-include macros: macroName:multiexcerpt-include
