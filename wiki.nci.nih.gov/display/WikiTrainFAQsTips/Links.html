When you create a link that points to a wiki page, the best type of link to create depends on where you intend to create that link:
  • If you intend to create the link within the wiki, refer to Linking from a Wiki Page to a Wiki Page .
  • If you intend to create the link outside of the wiki, refer to Linking from Another Website to a Wiki Page .
When you create a link within the wiki that points to another website, refer to Linking from a Wiki Page to Another Website . This page family includes the following:
  • Linking from a Wiki Page to a Wiki Page
  • Linking from Another Website to a Wiki Page
  • Linking from a Wiki Page to Another Website
  • Marking Links as External to US Government
  • Linking to a Specific Place on a Page
  • Putting a List of Child Pages on a Page
  • Putting a Table of Contents on a Page
  • Listing Links So Only Users with Permission Can View Them
Also refer to the following pages:
  • Linking from an Image to Another Page
  • Uploading and Linking to an Attachment
  • Atlassian help page: Links
Key point for accessibility: Each link on a page must be unique.
