To embed a Google Calendar in a wiki page, use the Google Calendar macro:
  1. Type { and the first few letters of the macro name, as follows: {goog
  2. When the macro browser opens, select Google Calendar.
  3. Select the macro panel and click Edit.
  4. In the Calendar URLs field, enter the URL of the feed for the calendar.
For instructions, refer to the following Atlassian plugin page: https://marketplace.atlassian.com/apps/231/google-calendar-for-confluence
