Group Data with the Group Data Macro

The group-data macro allows data to be grouped within a named data set. This replaces the scaffolding macro. This page shows the general form and an example. More information The general form is {group-data:name} {xxx-data:sub-field} {group-data} An example follows. {group-data:MeetingInformation} {text-data:NextMeeting|type=line}{text-data} {date-data:MeetingDateTime|format=yyyy-MMM-dd HH-mm-ss}today{date-data} {group-data} The group data macro establishes the group. The text data macro establishes the field name, Next meeting:. The date data macro establishes the day and time, initially the date the macro was created with the time not specified. In the Edit Contents view, select a different date and enter the time.Next meeting:2014-Dec-11 11-00-00Editing in the Edit Contents view suppresses output of macros entered in earlier versions of Confluence. Line breaks between the field name and date and time are not rendered.
