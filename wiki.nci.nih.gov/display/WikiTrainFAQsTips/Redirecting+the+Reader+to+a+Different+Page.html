You can automatically redirect the reader to another page. For examples, try searching for the Redirect macro .To view or edit a page which redirects instantly due to the Redirect macro, add '?redirect=false' to the URL manually.To create the redirection
  1. In the page you want to change, clickÂ Edit.
  2. Type as much of the following as needed to open the macro browser. {redirect
  3. Select the Redirect macro. The Insert 'Redirect' Macro dialog box appears.
  4. Enter the URL of the page to which you want to redirect.
  5. Consider entering a delay. A minimum of 5 seconds would make it easier for someone to edit the page.
  6. Click Insert.
For more information, refer to the following plugin help page: https://docs.servicerocket.com/redirection/macro-reference/redirect
