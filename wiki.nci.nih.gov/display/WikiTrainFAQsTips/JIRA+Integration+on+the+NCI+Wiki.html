Integration with JIRA provides these advantages:
  • You can log in on either the NCI Wiki or Tracker and you will be logged in on both sites.
  • If you have editing permissions on the wiki:
    • You can use a shortcut to display any one JIRA issue. Refer to Using the JIRA Shortcut .
    • You can display a list of issues on a wiki page. Refer to Using the JIRA Macro .
    • You can display a chart on a wiki page that shows selected data about JIRA issues. Refer to Using the JIRA Chart Macro .
    • If the default assignee in tracker.nci.nih.gov is configured as “unassigned” in the project, you can use the JIRA macro to create JIRA issues on the NCI Wiki. Refer to Using the JIRA Macro .
  • You can add dates for JIRA events, sprints, and releases on a Team Calendar in Confluence. Key point for accessibility: Because a Team Calendar has an image that is invisible and has no alt text, only essential calendars should be embedded in a public wiki page. Refer to the following Atlassian help page: https://confluence.atlassian.com/teamcal/add-jira-events-241566244.html
Improvements in JIRA integration beginning with Confluence 3.5 are shown in the demonstration provided by Atlassian: http://www.atlassian.com/en/software/confluence/whats-new-iframe/35#3.-better-jira-integration
