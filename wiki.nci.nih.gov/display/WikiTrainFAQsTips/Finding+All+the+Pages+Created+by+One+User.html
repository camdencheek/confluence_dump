To find all of the pages created by a particular user, you have the following options:
  • Perform a Confluence Full Search for those pages and filter by creator ( example ), as described in the following Atlassian help page: https://confluence.atlassian.com/conf613/search-964961041.html
  • Use the Content by User macro to list all the pages created by one user: https://confluence.atlassian.com/conf613/content-by-user-macro-964960798.html The page displays a table listing all the pages created by the user.
