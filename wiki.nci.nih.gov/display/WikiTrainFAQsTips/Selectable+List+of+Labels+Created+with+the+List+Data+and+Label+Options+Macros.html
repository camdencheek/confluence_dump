The list data and label options macros generate a selectable list of labels which may be applied to the current page. More information on the label options macro The general form is {list-data:[Available Labels]} {label-options:[labeloptiontest1, labeloptiontest2]} {list-data} An example follows.The list data macro establishes a selectable list. The label options macro establishes a list of labels. In the Edit Contents view, select the label to apply to the page. The most recently selected label is listed by label options macro and applied to the page.labeloptiontest1
