Form Fields Created with Scaffolding Macros

This section explains the following macros.
  • Date Data macro
  • Text Data macro
More information about the text data macro

Date Data macro

The general form is {date-data:Name|format=[date format]}[<FIXED DATE>, yesterday, today, tomorrow, now]{date-data} An example follows. {date-data:MeetingDateTime|format=yyyy-MMM-dd HH-mm-ss}today{date-data} The date data macro establishes the date, initially the date the macro was created. In the Edit Contents view select a different date.2011-Apr-04 00-00-00

Text Data macro

The text-data macro allows entering and editing free text in the Edit view.The general form is: {text-data:Name|type=[line, area]}Default text.{text-data} An example follows. Username: {text-data:Username|width=100px}{text-data} Password: {text-data:Password|type=password|width=100px}{text-data} || Address: | {text-data:Address|type=area|width=400px|height=150px}Street, City, State, Postcode Country{text-data} | The text data macros establish the Username, Password, and Address fields. In the Edit Contents view, enter the username, password, and address.Username: Password:
Address: Street, City, State, Postcode Country
