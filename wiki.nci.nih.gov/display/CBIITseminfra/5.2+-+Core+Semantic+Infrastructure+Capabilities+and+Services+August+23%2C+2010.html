This section describes all the core semantic infrastructure capabilities and services. The services are organized by functional profiles that aggregate the core services and capabilities. The profiles described in this section address the following requirements:
  • Artifact Management
  • Service Discovery and Governance
  • Forms Modeling
  • Conformance Testing
  • Terminology and Platform Integration
The functional profiles organized by the requirements are as follows:Artifact Management
  • Analyze Artifacts
  • Administer Artifacts
  • Search and Access
  • Transform
  • Model and Annotate
Service Discovery and Governance
  • Analyze Services
  • Administer Services
  • Search and Access Services
  • Service Governance and Workflows
  • Access Control and Policies
Forms Modeling
  • Administer Forms
  • Search and Access
  • Create Forms
Conformance Testing
  • Test for Conformance
  • Search and Access Conformance Statements
Platform and Terminology Integration
  • Search
  • Access
The search and access profiles for each requirement are different as they are customized to access specific information in a defined format. For example, platform tools like the orchestration engine will leverage specific metadata to determine compatibility between two services.
