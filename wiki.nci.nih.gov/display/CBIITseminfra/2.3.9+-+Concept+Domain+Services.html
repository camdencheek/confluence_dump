The following table shows the functions of CTS2 for concept domain mangement.
CTS2 Function Concept Domain Management
API must return the list of available Concept Domain Names in the system. List Concept Domains
API must return the list of Value Sets that are bound to a given Concept Domain. List Concept Domains
API must return the list of Usage Contexts that are bound to the given Concept Domain. List Concept Domains
API must return the list of Metadata attributes/properties of the given Concept Domain List Concept Domains
API must return the details of a given Concept Domain. Return Concept Domain Details
API must return the list of UsageContexts that are available in the system List Usage Contexts
API must return the details of a given Usage Context. Return Usage Context Details
API must return the list of Concept Domain Bindings that are available in the system. List Concept Domain Bindings
API must return whether or not a given Concept has an association to a given Concept Domain. Check Concept to Concept Domain
API must allow the creation of a Concept Domain Object. Create Concept Domain
API must allow for a Concept Domain to be updated/maintained. Maintain Concept Domain
API must allow for the creation of an Entity designated with a "Usage Context Code System" type. Create Usage Context
API must allow for a Usage Context to be updated/maintained. Maintain Usage Context
