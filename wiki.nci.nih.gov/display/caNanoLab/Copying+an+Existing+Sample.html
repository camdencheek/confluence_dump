Copying an existing sample copies all of the annotation and/or publications, making it easier to create and submit similar samples.To copy an existing sample
  1. Click the Samples menu. The Manage Samples page appears.
  2. Click Copy an Existing Sample. On the Copy Sample page, both fields are required.
  3. In Existing Sample, specify a complete or partial sample name.
  4. From the returned list, select the sample that you want to copy.
  5. Specify the New Sample Name.
  6. Click Submit. An Update Sample page appears with the new sample name and the same information as the original sample.
  7. Edit the copied sample as needed.
  8. Click Update.
