In a characterization, you can add results in the form of supporting documents, such as a graphic, spreadsheet, or images.You can add as many files as you wish.To add a file to a characterization
  1. Access a sample characterization, as described in Accessing the Sample Characterization Summary .
  2. Next to the characterization of interest, click Add or Edit.
  3. Next to Finding, click Add.
  4. In the Finding section of the characterization, next to Files, click Add.
  5. Select Upload or Enter File URL.
    1. If you select Upload: Click Choose File. Browse and select the file (up to 30 MB).
    2. If you select Enter File URL: Enter the URL where the document is located.
  6. Complete the following.
    1. Select the File Type (required), Document, Graph, Image, Movie, or Spreadsheet.
    2. Enter the File Title (required).
    3. Specify Keywords to associate with the file
    4. Enter a Description of additional information of the file.
  7. Click Submit to add the file(s) to the sample.
