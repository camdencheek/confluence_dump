The following table lists and describes each caNanoLab menu.
Menu Options Description
Home Returns to the home page.
Workflow Presents a workflow graphic with active links to access caNanoLab functions. For details, refer to Using the Workflow to Get Started
Protocols Search for existing protocols, create a new protocol, and submit a protocol for review for public access. For details, refer to Managing Protocols in caNanoLab .
Samples Search for existing samples , create a new sample, and submit a sample for review for public access, as described in Managing Samples in caNanoLab . When you add a sample, the following options appear in the Navigation Tree:
NAVIGATION TREE
GENERAL INFO
COMPOSITION
CHARACTERIZATION
PUBLICATION
  • Composition to add annotations and publications to the sample. For details, refer to Managing Sample Composition Annotations .
  • Characterization to review or ascribe characterizations to samples. For details, refer to Managing Sample Characterizations .
  • Publication to add publication to a sample. For details, refer to Adding Publications to a Sample .
Publications Search for sample publications, create a new publication, or submit a publication for review for public access. For details, refer to Managing Publications in caNanoLab .
Groups Create and manage collaboration groups which allow researchers to share data among members of the group.
  • Curators can manage all collaboration groups and view a list of samples assigned to each group.
  • If you do not have the curator role, you can manage only your own collaboration groups.
For details, refer to Managing Collaboration Groups .
Curation Curator only: Review items pending public release and generate sample data metrics in a batch. For details, refer to Managing Data Curation .
Results Curator only: Appears after you perform a batch process. For details, refer to Reviewing Batch Results .
My Workspace Review the samples, protocols, and publications that you saved or are associated with your log in. For details, refer to Using My Workspace .
My Favorites Review the samples, protocols, and publications that you bookmarked for easy access. For details, refer to Using My Favorites .
Admin Admin only: Add and edit users, and reset user passwords. For details, refer to Managing caNanoLab User Accounts .
Login/Logout Login or out of caNanoLab.
