Primary features of caNanoLab release 1.5.2 include:
  1. Migration to LDAP-based user authentication
  2. Support for data submission by individual researchers
  3. Support for collaboration groups
  4. Support for read/write data access for collaboration groups and individual users
  5. Support for data availability metrics against the caNanoLab model and the Minimum Information on Nanoparticle Characterization (MINChar) standard
  6. Support for data ownership transfer
